### Title
Unvalidated Feed-to-Token Binding in `AnchoredProviderFactory.createAnchoredProvider` Allows Misbound Provider to Reach Pool Swaps — (`smart-contracts-poc/contracts/AnchoredProviderFactory.sol`)

---

### Summary

`AnchoredProviderFactory.createAnchoredProvider` is permissionless and validates the oracle allowlist and envelope parameters, but never validates that `baseFeedId`/`quoteFeedId` correspond to the `baseToken`/`quoteToken` pair supplied by the caller. Because the oracle layer is explicitly "token-free," no on-chain registry can close this gap. Any caller can deploy a recognized provider (`isProvider() == true`) whose `token0()`/`token1()` match a real pool's tokens but whose feed prices a completely different asset pair. If that provider is attached to a pool, every swap executes at the wrong oracle price.

---

### Finding Description

`AnchoredProviderFactory.createAnchoredProvider` enforces two classes of checks before deploying an `AnchoredPriceProvider`:

1. **Oracle allowlist** — `_oracles.contains(oracle)` must be true.
2. **Envelope bounds** — `minMargin`, `maxRefStaleness`, `maxSpreadBps` must lie within the admin-configured envelope for the feed's class. [1](#0-0) 

What is **never** checked is whether `baseToken`/`quoteToken` correspond to the asset pair priced by `baseFeedId`/`quoteFeedId`. The constructor comment is explicit that the oracle layer carries no token information: [2](#0-1) 

The `AnchoredPriceProvider` stores `baseToken` and `quoteToken` as immutables and exposes them via `token0()`/`token1()`, but these values are never cross-referenced against the feed: [3](#0-2) 

The price read path calls `_readLeg(baseFeedId)` which fetches `price(baseFeedId, pool)` from the oracle — entirely independent of `baseToken`: [4](#0-3) 

The factory's own eligibility predicate for public pools is `isProvider(p)`, which returns `true` for every provider deployed through `createAnchoredProvider`: [5](#0-4) 

**Attack path:**

1. Attacker calls `createAnchoredProvider` with:
   - `oracle` = any admin-allowlisted oracle (e.g., Pyth)
   - `baseFeedId` = ETH/USD feed ID (a valid, existing feed)
   - `baseToken` = WBTC address, `quoteToken` = USDC address
   - Envelope-valid `minMargin`, `maxRefStaleness`, `maxSpreadBps`
2. Factory deploys the provider; `isProvider()` returns `true`.
3. A WBTC/USDC pool is created (or updated) with this provider. The pool factory's token-alignment check (`provider.token0() == WBTC`, `provider.token1() == USDC`) passes because the attacker supplied the correct token addresses.
4. Every call to `_getBidAndAskPriceX64()` in the pool fetches ETH/USD prices and uses them to price WBTC/USDC swaps: [6](#0-5) 

5. Traders buy WBTC at ETH/USD price (~$3 000 instead of ~$60 000); LPs lose the difference.

The analog to the external bug is exact: just as any ERC-20 could be deposited into a bounty with no token whitelist, any feed ID can be bound to any token pair with no feed-to-token registry. In both cases the unvalidated "foreign input" corrupts the core settlement path.

---

### Impact Explanation

**Critical.** A pool executing WBTC/USDC swaps at ETH/USD prices allows traders to drain LP-deposited WBTC at a ~20× discount. LP principal loss is direct and immediate; the pool's accounting invariant (`binTotals.scaledToken0` tracks real WBTC but prices it as ETH) is permanently broken for the duration the misbound provider is active.

---

### Likelihood Explanation

**Medium.** `createAnchoredProvider` is permissionless; any address can deploy a misbound provider at zero cost. The remaining condition — that the misbound provider reaches a pool — depends on whether pool creation is permissionless (the factory design and the `isProvider()` eligibility predicate strongly imply it is) or requires an admin to set the provider via `setPriceProvider`. Even in the admin-gated case, the misbound provider passes every machine-checkable predicate (`isProvider()`, token alignment), so an inattentive admin could set it without detecting the feed mismatch.

---

### Recommendation

In `AnchoredProviderFactory.createAnchoredProvider`, require the caller to supply a feed-to-token attestation that the factory can verify, or maintain an admin-curated mapping `feedId → (baseToken, quoteToken)` and assert:

```solidity
(address expectedBase, address expectedQuote) = feedTokens[baseFeedId];
require(baseToken == expectedBase && quoteToken == expectedQuote, TokenFeedMismatch());
```

Alternatively, add a `setFeedTokens(bytes32 feedId, address base, address quote)` admin function (analogous to `setFeedClass`) and enforce the check inside `createAnchoredProvider` before deployment.

---

### Proof of Concept

```solidity
// Assume: pythOracle is allowlisted, ETH_USD_FEED_ID is a valid feed,
//         envelope for DEFAULT_CLASS is configured.
address misboundProvider = anchoredFactory.createAnchoredProvider(
    pythOracle,
    ETH_USD_FEED_ID,   // prices ETH/USD
    bytes32(0),        // single-feed
    minMargin,
    maxRefStaleness,
    maxSpreadBps,
    false,
    0,
    WBTC,              // token0 = WBTC  ← correct for the pool
    USDC               // token1 = USDC  ← correct for the pool
);

// isProvider() returns true — passes public-pool eligibility gate
assert(anchoredFactory.isProvider(misboundProvider));

// Pool created with WBTC/USDC tokens and this provider.
// provider.token0() == WBTC, provider.token1() == USDC → token check passes.
// But getBidAndAskPrice() returns ETH/USD Q64 prices.
// Every swap in the WBTC/USDC pool executes at ETH price, not WBTC price.
// Attacker swaps USDC → WBTC at ETH/USD rate, draining LP WBTC at ~20x discount.

        address oracle,
        bytes32 baseFeedId,
        bytes32 quoteFeedId,
        uint256 minMargin,
        uint256 maxRefStaleness,

        bool    mutableParams,
        int256  marginStep,
        address baseToken,

    ) external override returns (address provider) {
        if (!_oracles.contains(oracle)) revert OracleNotAllowed(oracle);

        // Feeds without an explicit class fall back to the admin-configured DEFAULT_CLASS envelope.
        bytes32 classId = feedClass[baseFeedId];
        if (classId == bytes32(0)) classId = DEFAULT_CLASS;

        Envelope storage env = envelopes[classId];
        if (!env.exists) revert EnvelopeNotFound(classId);
        if (
            minMargin < env.minMarginMin || minMargin > env.minMarginMax
            || maxRefStaleness < env.stalenessMin || maxRefStaleness > env.stalenessMax
            || maxSpreadBps < env.maxSpreadMin || maxSpreadBps > env.maxSpreadMax

        AnchoredPriceProvider p = new AnchoredPriceProvider(
            address(this),
            oracle,
            baseFeedId,
            quoteFeedId,

            maxRefStaleness,
            maxSpreadBps,
            mutableParams,
            marginStep,
            baseToken,
            quoteToken

        provider = address(p);
        address creator = msg.sender;

        _providers.add(provider);
        _providersByCreator[creator].add(provider);

    function _getBidAndAskPrice() internal returns (uint128, uint128) {
        (uint256 mid, uint256 spreadBps, , bool ok) = _readLeg(baseFeedId);
        if (!ok) return (0, type(uint128).max);

        bytes32 _quote = quoteFeedId;
        if (_quote != bytes32(0)) {
            (uint256 mid2, uint256 spreadBps2, , bool ok2) = _readLeg(_quote);
            if (!ok2 || mid2 == 0) return (0, type(uint128).max);
            // Synthetic ratio (8-decimal): mid1 / mid2. Relative uncertainties of a ratio add.
            mid = Math.mulDiv(mid, ORACLE_DECIMALS, mid2);
            spreadBps += spreadBps2;
        }

        return _computeBidAsk(mid, spreadBps);
    }

    function _readLeg(bytes32 feedId)
        internal returns (uint256 mid, uint256 spreadBps, uint256 refTime, bool ok)
    {
        (mid, spreadBps, , refTime) = IPricedOracle(address(offchainOracle)).price(feedId, msg.sender);

        // Stale reference → not ok. Clamping to a stale anchor is the one false-safety case.
        if (_isStale(refTime, block.timestamp, MAX_REF_STALENESS)) return (mid, spreadBps, refTime, false);

        // Basic validity — mid positive, spreadBps not the stalled/off-hours marker (the Chainlink oracle
        // writes spreadBps = ORACLE_BPS when an RWA market is closed).
        if (mid == 0 || spreadBps >= ORACLE_BPS) return (mid, spreadBps, refTime, false);

        // Per-leg price guard.
        (uint128 guardMin, uint128 guardMax) = offchainOracle.priceGuard(feedId);
        guardMax = guardMax == 0 ? type(uint128).max : guardMax;
        if (mid < guardMin || mid > guardMax) return (mid, spreadBps, refTime, false);

        ok = true;
    }
