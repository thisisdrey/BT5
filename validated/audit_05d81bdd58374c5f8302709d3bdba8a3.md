### Title
Provider Owner Can Instantly Swap `IAnchorSource` to Deliver Unbounded Wide Quotes to Pool Swaps — (`smart-contracts-poc/contracts/AnchoredProviderFactory.sol`)

### Summary

`AnchoredProviderFactory.setSource` lets the provider owner replace the live `IAnchorSource` of any `AnchoredPriceProvider` instantly, with no timelock. The protocol's stated safety justification is "any source is clamp-bounded by the provider at all times." However, the clamp only prevents the source from quoting *tighter* than the anchor band — it does not prevent the source from quoting arbitrarily *wider* (bid → 1, ask → `type(uint128).max − 1`). Those wider quotes pass both the clamp and the `FeedStalled` sentinel check and reach pool swaps unchanged, causing direct loss of trader principal.

### Finding Description

`AnchoredProviderFactory.setSource` is gated only by `onlyProviderOwner`: [1](#0-0) 

The comment explicitly documents the absence of a timelock:

> "Swap a provider's source (zero → reference mode). The curator's only knob — **instant, no timelock**: any source is clamp-bounded by the provider at all times."

The clamp in `AnchoredPriceProvider._getBidAndAskPrice` enforces:

```
final_bid  = min(source_bid,  band_bid)   // source cannot quote HIGHER bid than band
final_ask  = max(source_ask,  band_ask)   // source cannot quote LOWER  ask than band
```

```text
    /// @notice Swap a provider's source (zero → reference mode). The curator's only knob — instant,
    ///         no timelock: any source is clamp-bounded by the provider at all times.
    function setSource(address provider, address newSource) external override onlyProviderOwner(provider) {
        require(_providers.contains(provider), ProviderNotTracked());
        AnchoredPriceProvider(provider).setSource(newSource);
        emit SourceSet(provider, newSource);
    }
```
