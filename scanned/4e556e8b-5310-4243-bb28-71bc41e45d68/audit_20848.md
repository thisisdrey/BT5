# Contract AutoStake lets anyone to set initial pool address, that can be exploited by setting immediately pool address after deploying/instantiating new contract.

## Summary
The bug report suggests that only the contract owner should be able to set the initial pool address. This means that other users should not have the ability to change or set the address for the initial pool. This recommendation is important because it can prevent unauthorized changes to the pool address, which could potentially lead to security issues.

## Details
**Recommendation**:
Allow only contract owner to set initial pool address.
