# Math could fail due to negative number in DVGToken.sol

## Summary
The bug report states that there is an issue with the function _moveDelegates at line 220. The variable srcRepOld can sometimes be equal to 0 or less than the amount, while line 221 expects it to be greater than or equal to the amount. The recommendation is to add a 'revert' or 'if' statement to properly handle the possibility of a negative value.
