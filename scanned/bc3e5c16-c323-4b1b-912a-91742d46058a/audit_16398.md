# Comparison operator used instead of assignment operator

## Summary
This bug report describes an issue with the `OracleUtilsLib.sol` file in the Portal contracts. The code contains a common typo where `==` is used instead of `=` in two places, resulting in incorrect storage updates. The recommended solution is to replace `==` with `=` in both instances. The code snippets are provided to make it easier to locate the typo.

## Details
#### Description


A common typo is present twice in the `OracleUtilsLib.sol` where `==` is used instead of `=` resulting in incorrect storage updates.


#### Examples


**code/contracts/Portal/utils/OracleUtilsLib.sol:L250**



```
self.\_validators[\_pk].state == 2;

```
**code/contracts/Portal/utils/OracleUtilsLib.sol:L269**



```
self.\_validators[\_pk].state == 3;

```
#### Recommendation


Replace `==` with `=`.
