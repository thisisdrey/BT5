# FR-1 | Cannot Withdraw Fees

## Summary
This bug report deals with the `FeeReceiver` contract, which is responsible for receiving fees for the entire protocol. The problem is that the contract does not have any functions to retrieve either erc20 tokens or ether, so any fees sent to the `FeeReceiver` contract are lost.

The recommendation is to add a withdraw function with proper access controls to withdraw and manage fees. The GMX Team has implemented this recommendation.
