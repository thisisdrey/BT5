# Change accessibility specifier of mint function

## Summary
This bug report is about a problem with the $JUST token's minting function. The white paper for the token stated that there would be a fixed amount of tokens, but the minting function does not have the correct accessibility modifier. This means that anyone, even outside of the contract, can call the function and potentially create more tokens. This goes against what was promised in the white paper and needs to be fixed.

## Details
**Description**

As stated in the white paper that there would only be a fixed amount of $JUST token, the
minting function should have a private accessibility modifier, so that after using it in the
constructor, no other contract or person can call it outside the contract.
