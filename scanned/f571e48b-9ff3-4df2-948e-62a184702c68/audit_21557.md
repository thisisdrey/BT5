# Remove debugging code from the repository ✓ Fixed

## Summary
This bug report is about source code from the development stage that was used for debugging the functionality and was not removed. This should not be present in the source code and even if they are used while functionality is developed, they should be removed after the functionality was implemented. Examples of the debug code can be found in the report. The issue was fixed by the development team. The recommendation is to remove the debug functionality at the end of the development cycle of each functionality.

## Details
#### Resolution
