# The check can be manipulated

## Summary
This bug report concerns a platform manager who can mistakenly update shares in the NFT contract and call `takeClaimingSnapshot`. This can lead to a situation where the athlete must add more funds to the distribution before closing it. To fix this issue, the `closeDistribution` function should be changed. This bug report is important for platform managers to be aware of in order to prevent any accidental updates or calls to the NFT contract.

## Details
##### Description
