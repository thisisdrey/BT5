# [H-03] Theft of ETH of free floating SLOT holders

## Summary
This bug report is about a malicious user being able to steal all claimable ETH belonging to free floating SLOT holders. The proof of concept can be found in a gist and can be tested with the forge tool. The recommended mitigation steps include using the += operator instead of = in the code found at a certain line of the Syndicate.sol file, as well as updating the state after each operation instead of relying on a lazy call to updateAccruedETHPerShares.
