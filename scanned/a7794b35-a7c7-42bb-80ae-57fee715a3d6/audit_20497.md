# No way to remove the merchant

## Summary
This bug report is about a problem with merchants in a mapping system. The report states that merchants can be added, but there is no way to remove their status as a merchant. The recommendation is to either confirm that disabling merchants is not possible or to add a way to remove their status.

## Details
**Description**
Merchants (from merchants mapping) can be added, but there is no functionality to remove
merchant status.
**Recommendation**:
Confirm, that merchant disabling is impossible or add removal method.
