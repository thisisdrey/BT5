# [H-10] Changing NFT contract in the MochiEngine would break the protocol

## Summary
Jonah1005 has identified a vulnerability in MochiEngine that could break the entire protocol. If the operator changes the NFT contract, all the vaults would point to a different NFT address and users would not be able to access their positions. The proof of concept can be found in MochiEngine.sol#L91-L93. As this is a high-risk issue, the recommended mitigation step is to remove the function.
