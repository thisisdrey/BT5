# [M-20] Auction multiplier set to zero

## Summary
This bug report is regarding a vulnerability in the function setAuctionMultiplier() from the code. The vulnerability allows the factory owner to set the auction multiplier to zero, which would stop the auction from settling and cause the function to revert. This vulnerability can be exploited by causing a safe math error, resulting in newRatio to revert. The proof of concept and tools used are not provided in the report. The recommended mitigation steps are also not mentioned.
