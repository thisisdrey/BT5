# [M-03] `setAdjustor` selector not included from `SimplexFacet`

## Summary
The `SimplexFacet` contract has a missing function called `setAdjustor` which cannot be called. This affects the contract's functionality. To fix this, the `simplexSelectors` array needs to be updated to include the `setAdjustor` selector.

## Details
## Severity

**Impact:** Medium

**Likelihood:** Medium

## Description

The `SimplexFacet` contract is missing the `setAdjustor` selector from the `simplexSelectors` array. This means that the `setAdjustor` function cannot be called.
