# Potentially `approved` cache miss

## Summary
This bug report is about a contract called AbyssSafe3.sol located at https://github.com/abyssfinance/abyss-lockup/blob/8fe1a854a9b01dc1aa35272b82fd22655d4f42d1/contracts/AbyssSafe3.sol#L202-L216. This contract has an approval caching mechanism that works fine if the token supports infinity approval. However, if the token does not support infinity approval, the cached approve might tell wrong information. 

It is recommended to check the approval permanently to ensure that the correct information is being given. This can be done by periodically checking the approval status to make sure that it is up to date. This will help to ensure that the correct information is being given and that the contract is functioning properly.
