# Incorrect view value

## Summary
The bug report is about a function called rewardPerShft() in a file called ShyftStaking.sol. The function is supposed to return a value of 0 if rewards have not been distributed yet and if the last update time is equal to 0. This is important because it relates to a specific period when pre-purchasers are added and rewards have not yet been distributed. The recommendation is to fix the function so that it works correctly.
