# [M-02] Users Can Always Grief the Protocol

## Summary
The bug report states that there is a medium risk issue where users can reduce the fees that go to the protocol by using a specific referral address. This means that the amount of fees collected by the protocol will be reduced and the reduced amount will not be accessible anymore. The team recommends redirecting fees to referrals only if they have some involvement with the protocol. The team has acknowledged the issue.

## Details
## Severity

Medium Risk

## Description

Users can reduce the amount of fees that go to the protocol by always specifying a referral address that is not able to claim these fees. Such addresses could be `address(0xdead)`, etc.

## Impact

Protocol fees will be reduced and the reduced amount will not be accessible anymore.

## Recommendation

Consider redirecting fees to referrals only if they have some input with the protocol, either staking or other action.

## Team Response

Acknowledged.
