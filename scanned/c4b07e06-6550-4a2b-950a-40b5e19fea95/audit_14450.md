# Mismatched Variable Names

## Summary
This bug report is about an error that occurs when users try to add except() arguments on function decrease\_1in examples/pausable-examples/pausable\_base/src/lib.rs line 49-52. The Rust compiler is unable to compile this code, and the recommendation is to change the variable check\_paused to \_\_check\_paused in the function if\_paused. This will ensure that the same variable is used in both the get\_bypass\_condition and if\_paused functions.
