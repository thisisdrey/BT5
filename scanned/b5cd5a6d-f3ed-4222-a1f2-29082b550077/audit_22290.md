# Price function is not linear

## Summary
This bug report is about a code that is intended to produce a linear price function, but instead produces a price curve with four steps of decreasing price. The initial period of the curve has 1 ether equal to 125 ARC. It is recommended that the code be reviewed to ensure that the price curve meets the desired shape.

## Details
The code seems to intend a linear price function, but this is the actual behavior:
