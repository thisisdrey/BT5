# `MintStarted`modifier missing

## Summary
This bug report is about the `MintStarted`modifier being missing in the `mintTo`function, which meant that the owner of the function was able to mint even when the minting operation was paused. To fix this, the recommendation was to add the `MintStarted`modifier to the `mintTo`function. This issue has been resolved.

## Details
**Description:** 

`MintStarted`modifier is missing meaning the owner can mint even when minting operation is paused.

**Recommendations:** 

Add `MintStarted`modifier in the `mintTo`function.

**Status:** Resolved.
