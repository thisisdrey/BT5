# Different Solidity versions.

## Summary
Bug Report Summary:

The bug report states that different pragma directives are used in a project, including interfaces. This can lead to vulnerabilities and is recommended to use the same directives throughout the project. The issue is classified as medium severity. The recommended solution is to use the same directives for the entire project.

## Details
**Description**

Different pragma directives are used. Throughout the project (including interfaces). Version
used: '=0.5.16', '>=0.5.0'
Issue is classified as Medium, because it is included to the list of standard smart contracts’
vulnerabilities.

**Recommendation**:

Use the same pragma directives for the entire project.
