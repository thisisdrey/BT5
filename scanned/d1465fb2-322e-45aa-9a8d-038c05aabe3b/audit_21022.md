# Unbounded loops in LiquidityMining ✓ Fixed

## Summary
This bug report is about a problem with some methods in the code/contracts/LiquidityMining.sol file. These methods contain unbounded loops which will fail when too many items are added to the arrays. The recommendation is to add limits (from, to) when requesting the items, and the bug has been fixed by doing this.

## Details
#### Resolution



Fixed by adding the limits.


#### Description


There are some methods that have unbounded loops and will fail when enough items exist in the arrays.

#### Recommendation
