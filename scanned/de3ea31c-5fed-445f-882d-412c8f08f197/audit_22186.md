# Fiat investments do not respect limits

## Summary
This bug report is about the Investments submitted by the admin via the `investFIAT` function not respecting the `maxRaiseWei` limit. The `maxRaiseWei` limit is an important feature of the `investFIAT` function as it sets the maximum amount of funds that can be raised. To solve this issue, the bug report suggests adding a `require(raisedWei &lt;= maxRaiseWei)` statement at the end of the function. This statement will ensure that the `maxRaiseWei` limit is enforced and that the amount of funds raised does not exceed the limit. The report has been updated with the `96e4c50` function which should help to resolve the issue.

## Details
Investments submitted by the admin via [`investFIAT`](https://github.com/fiinu/smart-contract/blob/7589b9d137bbf902e142e4f0476b06976e390bdb/FiinuCoin.sol#L246) do not respect the `maxRaiseWei` limit. Consider enforcing this by adding `require(raisedWei &lt;= maxRaiseWei)` at the end of the function.
