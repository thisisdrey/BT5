# Escrows can be reused

## Summary
The bug report states that there is an issue with the escrow feature where it can be reused multiple times. This means that the feature allows for multiple uses, but there is no guarantee that it will work correctly each time. The recommendation is to limit the usage of the escrow feature to only one time to avoid any potential issues.

## Details
##### Description
Currently, all escrows can be reused. However, the logic of a condition cannot guarantee that it will be correct for two consecutive executions.

##### Recommendation
We recommend restricting the usage of the escrows more than one time.
