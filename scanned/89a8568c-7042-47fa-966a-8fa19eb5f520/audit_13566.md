# Claiming could be blocked

## Summary
This bug report identifies an issue with the FantiumClaimingV1.sol contract. Users who have already claimed before the manager calls the function will have to wait until the athlete transfers tokens. The recommendation is to call the `addDistributionAmount` in the `updateDistribtionTotalEarningsAmounts()` and `updateDistributionEventCollectionIds` functions. This will ensure that users do not have to wait for the athlete to transfer tokens before they can claim.

## Details
##### Description
Users that were able to claim before the manager calls this function https://github.com/FantiumAG/smart-contracts/blob/c622b4c35132167bacc22cf76656d80715edb5fd/contracts/claiming/FantiumClaimingV1.sol#L316-L337 will have to wait until the athlete transfers tokens.

##### Recommendation
We recommend calling `addDistributionAmount` in the `updateDistribtionTotalEarningsAmounts()` and `updateDistributionEventCollectionIds` functions.
