# TransferFrom from `address(this)` will not work for most of the tokens

## Summary
This bug report is about a smart contract written in Solidity, a programming language used for writing Ethereum smart contracts. The issue is that the contract uses `transferFrom(address(this),...)` which will not work with most of the tokens without calling `approve`. The recommendation is to use `SafeTransfer` instead of `TransferFrom`.
