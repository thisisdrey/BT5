# User can't repay with force flag

## Summary
The bug report describes an issue where users are unable to close their account if a token has been blocked. This is because they must wait for someone else to liquidate the account. The report recommends adding a new parameter to the repay function to allow users to force the account to be closed.

## Details
##### Description
If some token was blocked on account, user can't close this account and must wait until account would be liquidated by someone:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/credit/CreditManager.sol#L368
##### Recommendation
We recommend adding a parameter for the repay function, so that user can repay account with force flag.
