# Error in fee calculation.

## Summary
The bug report is about a mistake in the calculation of a variable called amountInPostFee in a file called ImpossibleLibrary.sol. The mistake occurs on line 112 and is related to a power of 10000 and a regular subtraction. The result is a loss of accuracy. The recommendation is to fix the calculation mistake.

## Details
**Description**

ImpossibleLibrary.sol, Line 112
amountInPostFee = amountInPostFee.sub(sqrtK.sub(reserveIn));
- amountInPostFee is powered by 10000
- sqrtK.sub(reserveIn) - regular.
There is a calculation mistake with the loose of accuracy

**Recommendation**:

Fix the calculation mistake
