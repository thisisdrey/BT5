# Maximum XVS Staked Does Not Match the Documentation

## Summary
The bug report points out an inconsistency in the documentation for a maximum XVS cap in the Venus Protocol. While the documentation states that the maximum cap is 100,000 XVS, the actual code sets the maximum cap to 10,000 XVS. The report suggests either updating the code or the documentation to match each other. The issue has been resolved in a recent pull request and commit.

## Details
The documentation states:



> The maximum XVS cap taken into account when calculating the score of a user is 100,000 XVS.


However the constant `[MAXIMUM_XVS_CAP](https://github.com/VenusProtocol/venus-protocol/blob/f31a0543da039dab69112c6be3e36ea54959503b/contracts/Tokens/Prime/PrimeStorage.sol#L29)` is set to `10000`. Consider modifying the variable or updating the documentation accordingly.


***Update:** Resolved in [pull request #196](https://github.com/VenusProtocol/venus-protocol/pull/196)* at commit [860c959](https://github.com/VenusProtocol/venus-protocol/commit/860c9598465b0e67092f838e3c5ee2faf7c1b664).
