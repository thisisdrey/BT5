# Token name and symbol can be changed

## Summary
The setInfo() function in the token code allows for the name and symbol of the token to be changed even after it has been deployed. This is not recommended because it allows for the government to change the token's name and symbol at any time. To fix this, a requirement check should be added to prevent the setInfo() function from being called more than once.

## Details
**Severity**: High

**Status**: Acknowledged

**Description**

The setInfo() function can be used to change the name and symbol of the token even after the token has been deployed. This is not advisable because the gov can change the name and symbol of the token anytime.

**Recommendation**: 

Add a requirement check to prevent calling the setInfo() function more than once.
