# `priceFeeds` can't be changed

## Summary
The bug report states that the `priceFeeds` cannot be changed in the Gearbox protocol contracts. This is a problem because there is no way to update the `priceFeeds` if they become outdated. The report suggests adding a function to allow for changing the `priceFeeds`.

## Details
##### Description
`priceFeeds` can't be changed:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/oracles/PriceOracle.sol#L56
##### Recommendation
We recommend adding a function for changing `priceFeeds`.
