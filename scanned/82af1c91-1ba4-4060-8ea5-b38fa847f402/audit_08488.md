# ORDERS CAN BE CANCELLED WHILE CONTRACT IS PAUSED

## Summary
The cancelInv() function in the Passages.sol file is missing the whenNotPaused() modifier. This means that orders can be cancelled while the contract is paused, which goes against the purpose of pausable contracts. The BVSS score for this bug is 6.3, which is considered a moderate risk. The recommendation is to add the whenNotPaused() modifier to the cancelInv() function, which has been done in commit 87eda80a.
