# `migrateToken` in MarinateV2.sol allows admin to withdraw funds, potentially concerning community.

## Summary
The report is about a bug found in a smart contract called MarinateV2.sol. The bug allows the admin to have too much control over the funds, which could lead to potential issues such as withdrawing funds without the community's consent. The recommendation is to add a multisig feature to prevent this from happening. The bug has been resolved by removing a specific function.
