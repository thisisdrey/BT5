# Allowed zero `borrowFresh` and `repayBorrowFresh`

## Summary
This bug report is about the functions `borrowFresh` and `repayBorrowFresh` not changing the state of the borrow balance. These functions only update the borrower's `borrowIndex`, consume gas and emit an "empty" event. The recommendation to fix this bug is to quit `borrowFresh` and `repayBorrowFresh` after checks. This can be done by adding the following code to the functions: `if (borrowAmount == 0)` and `if (repayAmount == 0)`. This code will update the `borrowIndex` and return a `NO_ERROR` status code.
