# [M-12] Incorrect condition handling in `BunniSwapMath::_computeSwap()`

## Summary
The report states that there is a bug in the `BunniSwapMath::_computeSwap` function, specifically at line `BunniSwapMath#L134`. The issue is that the `if` statement does not properly check if `naiveSwapResultSqrtPriceX96` is equal to `sqrtPriceNextX96`. This causes the Liquidity Density Function (LDF) to use the wrong input when computing the swap, resulting in an incorrect swap calculation. The recommendation is to modify the condition at line `BunniSwapMath#L134` to include a check for `naiveSwapResultSqrtPriceX96 == sqrtPriceNextX96`.

## Severity
