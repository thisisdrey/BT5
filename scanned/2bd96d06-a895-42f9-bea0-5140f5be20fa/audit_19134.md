# Too many rights for configurator

## Summary
The bug report states that there is an issue with the Configurator tool, which has too much power and can potentially steal all of the user's funds. The link provided shows the specific code in the CreditFilter.sol file that is causing the problem. The recommendation given is to use a DAO (Decentralized Autonomous Organization) for the Configurator instead.

## Details
##### Description
