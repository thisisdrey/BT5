# Enable verification of Android Security Provider

## Summary
The Magma Wallet Android Client does not explicitly check if it is running on a device with an up-to-date Android Security Provider. This can leave the wallet open to network attacks, such as man-in-the-middle attacks, as the Security Provider is responsible for providing secure network communications. To mitigate this vulnerability, the ProviderInstaller.installIfNeeded() Google Play services should be run on every app startup to ensure the Android Security Provider is up to date. If the Security Provider remains out of date or an error occurs, Magma should decline to run. Additionally, anti-tamper and security update protections should be added to the Magma Wallet clients.

## Configuration Report

## Type

Medium

## Description
The Android client does not explicitly check whether it is running on a device that has an up-to-date Android Security Provider. The Security Provider is responsible for providing secure network communications, such as SSL/TLS. Running Magma Wallet on a device with an outdated Security Provider exposes it to network attacks. For example, it can allow an attacker on the network to decrypt and compromise SSL/TLS traffic.

## Exploit Scenario
A new vulnerability discovered in Android can be exploited to produce a man-in-the-middle attack (similar to [CVE-2014-0224](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0224)). Bob has not upgraded his phone to include the latest version of the Android Security Provider to mitigate this vulnerability, and his SSL traffic to the Magma API server can be snooped and modified.

## Recommendation
In the short term, on every app startup, run `ProviderInstaller.installIfNeeded()` from Google Play services. This method will ensure that the Android Security Provider is up to date. If the Security Provider remains out of date or an error occurs, this method will throw an exception and Magma should decline to run.

In the long term, continue adding anti-tamper and security update protections to the Magma Wallet clients.
