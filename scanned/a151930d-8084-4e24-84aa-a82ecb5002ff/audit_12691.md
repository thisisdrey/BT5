# H-6: vault balance not updated in withdrawReserves()

## Summary
This bug report is about a critical issue found by the Protocol Team in the D3Vault.sol file, which impacts almost all the funding related functions. The issue is that the vault balance is not being updated in the withdrawReserves() function. The Protocol Team used manual review to find this issue and the recommendation is to update the vault balance in the withdrawReserves() function. It is important to note that this issue is not part of the contest submissions and is not eligible for contest rewards.
