# Use ECDSA library.

## Summary
The report describes a bug in the CommonV1.sol file, specifically in the initNewRoundSigned() function. During testing with Hardhat, ethers, and web3, the signature verification did not work as expected. This caused the function to fail when trying to decode the r, s, and v parameters from the signature. The recommendation is to use the ECDSA function from the standard openzeppelin library to properly validate the signature. An example of how to use ECDSA is provided in the report.

## Details
**Description**


CommonV1.sol, function initNewRoundSigned().
Current verification of signature did not work as intended during testing with Hardhat, ethers
and web3. Function executing failed on decoding r, s, v parameters from signature.
