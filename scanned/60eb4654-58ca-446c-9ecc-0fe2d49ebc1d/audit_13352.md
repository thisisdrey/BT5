# Users' rewards can be lost if the incentive maker adds a new farming with the same key

## Summary
This bug report is related to the AlgebraEternalFarming.sol codebase. If the incentive maker adds a new farming after the previous one was unattached by the pool, then all users' rewards will be lost. This problem can be solved by adding a check that the maker cannot create a farming with the same key. This will prevent users' rewards from being lost.
