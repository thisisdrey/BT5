# Unused modifier

## Summary
This bug report is about a contract called ArgoPayments.sol that has two functions (charge and chargeWithProvider) that use a modifier called whenNotPaused. However, there seems to be an issue with this modifier as the _pause() function in another contract called Pausable.sol is set to false by default. The recommendation is to either add a new function called pause() to ArgoPayments.sol or to remove the whenNotPaused modifier and its import. This will fix the issue with pausing the functions in ArgoPayments.sol.

## Details
**Description**

Contract ArgoPayments.sol has two functions (charge/chargeWithProvider) that use a modifier
whenNotPaused. Functions _pause() in contract Pausable.sol is internal and it’s state by default
is false.

**Recommendation**:

Add new function pause() to the contract ArgoPayments.sol if charge/chargeWithProvider need
to be pausable or remove modifier whenNotPaused and its import.
