# Mistake in ENR key for discovery

## Summary
This bug report is about a function in the code for Ethereum protocol. The function is supposed to return the word "over" but instead, it is returning "over1". This is considered a medium risk issue and the recommendation is to change the return value to "over".

## Details
## Severity: Medium Risk

**Context:** `eth/protocols/eth/discovery.go#L36`

**Description:** This function will return "over1" when it should return "over".

**Recommendation:** Change the return value to "over".
