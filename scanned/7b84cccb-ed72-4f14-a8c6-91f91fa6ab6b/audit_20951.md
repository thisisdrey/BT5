# There is no input parameter processing in the method

## Summary
This bug report is about the `adjustPosition()` method in the contracts at the following lines:
- https://github.com/orbxball/stablecoins-3pool/blob/adeb776933c6cb3b8306239cc3357d4c6239a88d/contracts/StrategyDAI.sol#L99
- https://github.com/orbxball/stablecoins-3pool/blob/adeb776933c6cb3b8306239cc3357d4c6239a88d/contracts/StrategyUSDC.sol#L99
- https://github.com/orbxball/stablecoins-3pool/blob/adeb776933c6cb3b8306239cc3357d4c6239a88d/contracts/StrategyUSDT.sol#L99

The problem is that the `adjustPosition()` method has an input variable `_debtOutstanding`, but there is no processing of this variable in the body of the function. It is recommended to either add handling to the variable or remove this variable.
