# Set credit limit by pausing the guardian

## Summary
This bug report concerns the Compound protocol, which is used to set credit limits for users. The bug is that when the guardian is paused, it can set a new credit limit for users, which should not be possible. To address this issue, the developers recommend updating the checks in the "_setCreditLimit" function to ensure that it is not possible to set a new credit limit when the guardian is paused.

## Details
##### Description
Pausing the guardian can set a new credit limit for users with the credit limit.
https://github.com/ibdotxyz/compound-protocol/blob/8cd45803b48552e344e22be280c9e1c03ec8644a/contracts/Comptroller.sol#L1313
##### Recommendation
We recommend updating the checks in the "_setCreditLimit" function.
