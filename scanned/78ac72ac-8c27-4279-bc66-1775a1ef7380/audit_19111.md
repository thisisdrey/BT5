# `masterCreditAccount` remains uninitialized

## Summary
The report states that there is a bug in the `masterCreditAccount` function in the Gearbox protocol contracts. This bug allows anyone to initialize the function, even though it should only be accessible to certain users. The recommendation is to fix this bug by properly initializing the function.

## Details
##### Description
`masterCreditAccount` remains uninitialized, so anybody can initialize it:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/core/AccountFactory.sol#L90
##### Recommendation
We recommend to initialize `masterCreditAccount`.
