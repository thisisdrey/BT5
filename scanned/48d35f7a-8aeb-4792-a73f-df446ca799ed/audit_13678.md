# BTOK-1 | Dangerous Approve

## Summary
This bug report is about the `BaseToken` which is a token used in the Ethereum platform. The problem with the token is that it only exposes a dangerous `approve` function, which can be exploited by frontrunners. To fix this, it was recommended that an `increaseAllowance` function should be implemented, so that users can increase their allowances without risk of frontrunning. The Key Team then implemented the `increaseAllowance` function, thus resolving the issue.

## Details
**Description**

The `BaseToken` only exposes the dangerous `approve` function rather than an additional alternative `increaseAllowance` function.

**Recommendation**

Implement an `increaseAllowance` function so that users may increase their allowances without risk of frontrunning.

**Resolution**

Key Team: The recommended `increaseAllowance` function was implemented.
