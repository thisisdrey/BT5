# Seconds are used instead of days

## Summary
The bug report states that there is an error in the code related to the variable `duration`. Currently, this variable is not being used, but if it were to be used, it would use the wrong unit of measurement - seconds instead of days. The report suggests that the correct units should be used for `duration`.

## Details
##### Description
`duration` is currently not used, but if it were used, it would incorrectly use seconds instead of days: https://github.com/MetaLex-Tech/borg-core/blob/9074503d37cfa1d777ef16f6c88b84c98b4f54eb/src/implants/daoVetoGrantImplant.sol#L28.
