# Work with incorrect decimals

## Summary
The price feed in the Gearbox protocol can sometimes return a price with less than 18 decimals. This can cause issues with accurate pricing. To fix this, it is recommended to check the decimals of the price feed for each token.

## Details
##### Description
Price feed can return price with not 18 decimals:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/oracles/PriceOracle.sol#L56
##### Recommendation
We recommend to check decimals of the `priceFeeds[token]`.
