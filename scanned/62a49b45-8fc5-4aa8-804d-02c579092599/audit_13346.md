# Rates can be updated for a deactivated farming

## Summary
This bug report concerns the AlgebraEternalFarming.sol contract in the Algebra repository. Currently, the contract does not have a check that prevents the incentive maker from setting non-zero rates for a deactivated pool. The recommendation is to add a check that the farming was deactivated and that this case only allows setting zero rates.

## Details
##### Description
There is no check that the incentive maker can't set non-zero rates for a deactivated pool.
https://github.com/cryptoalgebra/Algebra/blob/7290fad656bfa89db3743c52af631154f6a8a2d5/src/tokenomics/contracts/farmings/AlgebraEternalFarming.sol#L198-L202

##### Recommendation
We recommend adding a check that the farming was deactivated and this case allows setting only zero rates.
