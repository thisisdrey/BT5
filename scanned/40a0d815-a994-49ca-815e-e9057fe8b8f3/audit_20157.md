# Organisation record can be re-written

## Summary
The MGD.sol code has a bug in the createOrganisation() function where the bool flag for the organisation name taken is not properly set. This makes the orgTaken mapping useless. The bug is marked as Medium severity because it can only be performed by an admin. The recommendation is to review the code and either remove the unused mapping or fix the storage setter.

## Details
**Description**

MGD.sol, createOrganisation()
Bool flag for the organisation name taken (orgTaken mapping) is never set. Thus, it makes
orgTaken mapping useless. The issue is marked as Medium only because the organisation
creation can be performed only by admin.

**Recommendation**:

Review the functionality, remove unused mapping or add storage setter.
