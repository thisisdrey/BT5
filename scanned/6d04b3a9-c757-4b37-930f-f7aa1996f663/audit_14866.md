# EXECUTOR has a full write access to the wallet storage

## Summary
The bug report describes a security issue with the EXECUTOR function in the EnsoWallet.sol contract. This function allows for write access to any storage slots through the executeShortcut function and delegatecall to a specific library. This can lead to unintended actions, such as transferring or renouncing ownership of the contract. The report recommends implementing access control for the DELEGATECALL function by creating a whitelist of permitted libraries. This will help prevent unauthorized access and potential exploitation of the contract.

## Details
##### Description
EXECUTOR has a write access to any storage slots vie the executeShortcut function and delegatecall to a specially crafted library. This allows to trigger
transfer/renounce of the OWNER address and other unintended actions.
https://github.com/EnsoFinance/shortcuts-contracts/blob/4902e55608f975f73772310955444110b1cfc4fc/contracts/EnsoWallet.sol#L57
##### Recommendation
We recommend implementing an access control for the DELEGATECALL, i.e. a whitelist of permitted libraries.
