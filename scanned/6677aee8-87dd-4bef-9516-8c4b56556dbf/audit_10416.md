# Missing Fallback Function to reject accidental Ether transfers to contract StableSwap_V2

## Summary
The bug report suggests that there is a problem with a contract that could result in Ether being stuck in the contract. The severity of the issue is medium and it has been resolved. The report recommends implementing a non-payable fallback function with proper logging to prevent this issue from happening in the future.

## Details
**Severity**: Medium

**Status**: Resolved

**Recommendations**:

Consider implementing a non-payable fallback function with appropriate logging to handle any unexpected Ether transfers to this contract in order to prevent Ether from being stuck in the contract.
