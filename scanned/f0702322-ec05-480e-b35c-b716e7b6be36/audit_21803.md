# [P1-M01] There are multiple hard-coded constants

## Summary
This bug report is about hard-coded values in the code which are prone to errors and hard to understand. Hard-coded values refer to booleans, numbers, and strings which are not defined as variables in the code. An example of hard-coded values is given in the report. The report suggests that for every hard-coded value, a constant variable should be defined with a clear and explanatory name. For the complex values, a comment should be added to explain how they were calculated or why they were chosen. The bug report was later updated to indicate that the hard-coded values have been fixed and moved to constants, commented, or removed.

## Details
###### Medium


Components: all
