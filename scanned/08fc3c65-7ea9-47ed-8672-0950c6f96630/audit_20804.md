# Variables used but not declared

## Summary
This bug report is about the Strategy contract in the Curve Voter Proxy repository. The Strategy contract uses address type variables such as `curve`, `gauge`, `reward` and `target` but there is no initialization procedure for these variables. This means that they are not given any values and can be used without having any values.

To fix this bug, it is recommended to initialize the local variables at declaration. This means assigning values to the variables when they are declared instead of waiting until they are used. This will help ensure that the variables have the correct values and can be used in the way they are intended.
