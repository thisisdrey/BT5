# Presale round does not return ETH

## Summary
The CryptoBearWatchClub.sol contract has a bug in the presaleMint() function where it does not return excess ETH like the mint() function does. This means that ETH may get stuck in the contract. It is recommended to check if presaleMint() should return excess ETH or fix the function's behavior.

## Details
**Description**

CryptoBearWatchClub.sol, presaleMint()
presaleMint() function does not return excess ETH amount like the mint() function does. Thus
ETH may be left stuck on the contract.

**Recommendation**:

Verify the presaleMint() should not return excess ETH or correct the behavior.
