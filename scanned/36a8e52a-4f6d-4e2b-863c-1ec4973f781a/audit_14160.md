# Dependencies With Publicly Known Vulnerabilities (Out of Scope)

## Summary
A bug report has been filed for the Snaps project, which defines a dependency within the yarn.lock file that is vulnerable to publicly known weaknesses rated as High or Medium in the CVSS scoring system. The identified weaknesses include Denial of Service, Regular Expression Denial of Service, Server Request Forgery, Open Redirect, and Insecure Credential Storage. It is recommended that the project review all identified dependencies and update them to the newest, stable version. Additionally, the project should review its current patch policy to ensure components are updated as soon as a fix exists. For the identified vulnerable components, specific versions have been identified that provide fixes.

## Details
### 3.2 Description


The snaps project defines a dependency (`@truffle/[[email protected]](/cdn-cgi/l/email-protection)` within the `yarn.lock` file vulnerable to publicly known weaknesses rated as `High` or `Medium` in the CVSS scoring system. It should be noted that the identified areas were not directly in the scope of the code review and are listed for the sake of completeness.


The following `@truffle/[[email protected]](/cdn-cgi/l/email-protection)` weaknesses were identified:


* Denial of Service `decode-uri-element` [CVE-2022-38900](https://nvd.nist.gov/vuln/detail/CVE-2022-38900) (CVSSv3 7.5)
* Regular Expression Denial of Service
* + `http-cache-semantics` [CVE-2022-25881](https://nvd.nist.gov/vuln/detail/CVE-2022-25881)(CVSSv3 7.5)
* + `cookiejar` [CVE-2022-25901](https://nvd.nist.gov/vuln/detail/CVE-2022-25901)(CVSSv3 7.5 - 5.3)
* + `ws` [CVE-2021-32640](https://nvd.nist.gov/vuln/detail/CVE-2021-32640) (CVSSv3 5.3)
* Server Request Forgery `request` [CVE-2023-28155](https://nvd.nist.gov/vuln/detail/CVE-2023-28155) (CVSSv3 6.5)
* Open Redirect `got` [CVE-2022-33987](https://nvd.nist.gov/vuln/detail/CVE-2022-33987) (CVSSv3 5.3)
* Insecure Credential Storage `web3` [SNYK-JS-WEB3-174533](https://security.snyk.io/vuln/SNYK-JS-WEB3-174533) (CVSSv3 3.3)
