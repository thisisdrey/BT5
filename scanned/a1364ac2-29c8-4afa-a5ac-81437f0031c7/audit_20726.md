# transfer() instead of call()

## Summary
The bug report is about a line of code in a file called EthDexManager.sol. The code in question, transfer() for Eth, is considered unsafe due to a recent update in the Ethereum system. The report suggests replacing this code with call() and adding restrictions for non-contract accounts.

## Details
**Description**

EthDexManager.sol, line 210
transfer() for Eth is considered as unsafe (because of Istanbul Ethereum update) and should
be replaced with call(). Especially because of missing restrictions to non-contract accounts.

**Recommendation**:

Change transfer() to call().
