# Restricted setter open for unauthorized users.

## Summary
The bug report is about a code upgrade that resulted in the deletion of a restricting modifier, which allowed anyone to call a specific function. This poses a security risk and the recommendation is to set the appropriate restricting modifier to the function. The bug has been resolved.

## Details
**Severity**: High

**Status**: Resolved 

**Description**

ChainportMainBride.sol
ChainportSideBride.sol
`setAddressLength()` setter can be called by anyone. The `onlyClassifiedAuthority` modifier
was deleted durin the code uprade.

**Recommendation**:

Set the appropriate restricting modifier to the function.
