# M-20: dust amounts will be removed in OFT tokens

## Summary
The report discusses a bug found by pashap9990 that affects the OFT tokens in the autonomint-judging project. The root cause of the bug is that dust amounts are being removed in OFT tokens, which impacts the core functionality of the contract. This is caused by a code in the CDSLib.sol file that removes decimal precision before cross-chain transfers. This can result in the protocol not having enough funds to cover withdrawal transactions. The suggested mitigation is to remove the dust amount before sending a message to other chains.
