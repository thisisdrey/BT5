# Event is probably missing

## Summary
This bug report is about the `CoverFlashBorrower` method `setFlashLender()` on line 134-137 of the CoverProtocol/cover-flashloan repository. Currently, this method does not emit an event when it is called. It is recommended to create an event called `NewFlashLender` to be emitted when this method is called. This would allow users to track the changes made to the `setFlashLender()` method and would help with debugging.

## Details
##### Description:
At the line https://github.com/CoverProtocol/cover-flashloan/tree/bea8cd444aa1791a97b2a81078dc887a835f7949/contracts/CoverFlashBorrower.sol#L134-L137
`CoverFlashBorrower`'s method `setFlashLender()` should probably emit an event.

##### Recommendation
It is recommended to create event: `NewFlashLender`.
