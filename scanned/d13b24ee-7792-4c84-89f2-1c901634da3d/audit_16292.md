# M-1: Curve LP Controller withdraw and claim function uses wrong signature

## Summary
This bug report concerns the Curve LP Controller withdraw and claim function, which uses the wrong signature. This issue was found by obront and was confirmed by zobront.

The function selector `0x00ebf5dd` is used for `WITHDRAWCLAIM` in both CurveLPStakingController.sol and BalancerLPStakingController.sol contracts. This selector corresponds to a function signature of `withdraw(uint256,address,bool)`, however, the `withdraw()` function in the Curve contract does not have an address argument, and instead the function signature reads `withdraw(uint256,bool)`, which corresponds to a function selector of `0x38d07436`.

As a result of this issue, users who have deposited assets into Curve pools will not be able to claim their rewards when they withdraw their tokens.

The code snippet can be found at https://github.com/sherlock-audit/2022-11-sentiment/blob/main/controller-merged/src/balancer/BalancerLPStakingController.sol#L30-L31. Manual Review was used as the tool to find this issue.

The recommendation for this issue is to change the function selector in both contracts to `0x38d07436`. The fix was implemented in https://github.com/sentimentxyz/controller/pull/48 and confirmed by zobront.

## Summary

The function signature used for `WITHDRAWCLAIM` in both CurveLPStakingController.sol and BalancerLPStakingController.sol are incorrect, leading to the function not succeeding.

## Vulnerability Detail

In both the CurveLPStakingController.sol and BalancerLPStakingController.sol contracts, the function selector `0x00ebf5dd` is used for `WITHDRAWCLAIM`. This selector corresponds to a function signature of `withdraw(uint256,address,bool)`.

```solidity
bytes4 constant WITHDRAWCLAIM = 0x00ebf5dd;
```

However, the `withdraw()` function in the Curve contract does not have an address argument. Instead, the function signature reads `withdraw(uint256,bool)`, which corresponds to a function selector of `0x38d07436`.

## Impact

Users who have deposited assets into Curve pools will not be able to claim their rewards when they withdraw their tokens.
