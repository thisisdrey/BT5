# Gas checks in Restore should be performed as soon as possible

## Summary
This bug report discusses a problem in the core/vm/evm.go file. The issue is related to gas checks, which are important for avoiding unnecessary work. Currently, the gas checks are not being done at the beginning of the function, which could lead to wasted effort. The report recommends moving the gas checks to the top of the function to prevent this issue. The severity of this bug is considered medium risk.

## Details
## Medium Risk Report

**Severity:** Medium Risk  
**Context:** core/vm/evm.go#L812-L834  
**Description:** Gas checks should be done as soon as possible to avoid unnecessary work. There are some operations, like a signature validation, prior to the gas checks. Additionally, if there were an error in these prior checks, it wouldn't cost the caller anything.  
**Recommendation:** Move the gas checks to the top of the function.
