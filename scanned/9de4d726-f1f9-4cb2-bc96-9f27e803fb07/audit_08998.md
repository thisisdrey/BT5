# riskPoolBalance is not used 

## Summary
The bug report is about a code for a project called QuailFinance. The code has a section (line 160) that takes a portion of the funds for risk balance. However, this balance is not being used and can only be retrieved by the owner through a specific process. This is not the intended use for the risk pool. The recommendation is to either remove the risk pool or complete the intended feature for which it was introduced.

## Details
## QuailFinance.sol Review

## Context
Location: QuailFinance.sol#L160-L160

## Description
For each winner picking, a portion of the funds is taken to account for risk balance. This balance is never used again, and the only way to retrieve it is for the owner to include it in the Merkle tree, which is not the intended use for the risk pool.

## Recommendation
Consider removing the risk pool or completing the feature for which the risk pool was introduced.
