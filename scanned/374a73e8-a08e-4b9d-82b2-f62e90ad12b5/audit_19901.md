# [M-02] Cannot claim reward

## Summary
A bug has been reported in the `ConcentratedLiquidityPoolManager.claimReward` function. This function requires `stake.initialized` to be set but it is never set. Additionally, it performs a strange computation of `128 - incentive.secondsClaimed` which will almost always underflow and revert the transaction.

The impact of this bug is that users are not able to claim rewards. To mitigate this issue, it is recommended to rethink how claiming rewards should work. The bug has been confirmed by sarangparikh22 (Sushi).

## Details
_Submitted by cmichel, also found by 0xsanson, broccoli, hickuphh3, and WatchPug_
