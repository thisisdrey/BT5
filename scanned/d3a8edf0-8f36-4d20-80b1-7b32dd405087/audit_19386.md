# EthGOVToken doesn’t mint tokens on creation

## Summary
The EthGOVToken has a bug where it does not create any tokens upon creation, resulting in a total supply of zero. This means that the token cannot be used for trading. The recommendation is to add a function to mint tokens in the constructor. The client has acknowledged this issue and plans to update the contract to include the mint token functionality.

## Details
**Description**

EthGOVToken doesn’t mint tokens on creation, it means that total supply will be equal to zero
and it will be impossible to use this token for trading.

**Recommendation**:

Add mint tokens in the constructor.

**Comment**:

The client said that this is correct logic for this moment and they will update this contract with
mint token functionality.
