# REW-1 | Chain Incompatibility

## Summary
The `withdrawTo` function is used in the `_transferAsETH` function when working with WETH on Arbitrum. However, this function is not supported on Avalanche C-chain or other networks that GMX may deploy on. The recommendation is to not use the `withdrawTo` function when deploying on Avalanche or other chains, and instead implement a method to receive Ether and relay it to the to address. The resolution to this issue is that the `withdrawTo` function has been replaced with `withdraw` by the Key Team.
