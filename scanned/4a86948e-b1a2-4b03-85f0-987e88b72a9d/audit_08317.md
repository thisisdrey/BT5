# MAIL SERVER MISCONFIGURATION

## Summary
The Truffles mail server had a security issue where an attacker could impersonate emails from Truffles and scam or defraud users, potentially causing financial and customer loss. This could also be used with another vulnerability to specifically target corporate users. The attacker was able to gain control of an account by tricking the victim into sending them an authentication token. The severity of this issue was rated as a 4 out of 10 for both impact and likelihood. Truffles has since resolved the issue and the targeted user did not receive the fraudulent email.

## Details
##### Description

Truffles mail server contained multiple misconfigurations, allowing an attacker to impersonate emails as Truffles and performing scams/fraud/phishing users, which may result in financial and customer loss.

Furthermore, an attacker could exploit the aforementioned vulnerability in conjunction with `(HAL-08)` to execute a specifically targeted attack against corporate users.

![The individual identified as the victim, with the email address alvaro.macias@halborn.com, received an email from the attacker, who had assumed the identity of admin@truffles.one.](https://halbornmainframe.com/proxy/audits/images/659ef559a1aa3698c0f13dfd){width="500" height="450"}  

Finally, the victim sends via telegram the token received in the authentication process to the attackers. The account was fully controlled by them.

Impact: 4

##### Recommendation

**SOLVED**: Truffles effectively resolved the issue. Consequently, due to the non-delivery of the email, the targeted user did not receive the message sent by the attacker.
