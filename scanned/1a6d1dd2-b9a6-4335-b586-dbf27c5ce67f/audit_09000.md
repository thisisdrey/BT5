# Claimed gas is locked forever 

## Summary
Bug Report Summary:

The QuailFinance.sol contract has a function called claimMyContractGas() that claims and deposits gas into the contract, but there is no way to withdraw this claimed amount. This could potentially lock the gas forever. The recommendation is to determine who should be able to claim the gas and implement a withdraw feature accordingly. This is considered a medium risk issue.

## Details
## QuailFinance.sol - Lines 213

## Description
`claimMyContractGas()` claims gas accumulated by the contract and deposits it into the contract itself. There is no functionality to withdraw this claimed amount, locking it forever.

## Recommendation
Consider who should be able to ultimately claim the allocated gas and implement a withdraw feature accordingly.
