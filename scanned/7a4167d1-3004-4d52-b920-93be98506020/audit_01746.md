# [M-04] ERC20 Transfer — Not All Tokens Return Boolean

## Summary
The report discusses a bug in a code that uses ERC20(token).transfer(...) to check if a transfer was successful. However, for certain tokens like USDT, the code fails to return any value, causing the check to fail and the transfer to revert even if it was successful. The team has acknowledged and fixed this issue.

## Details
## Severity

Medium

## Description

Uses ERC20(token).transfer(...) and checks whether it returns true to confirm success. However, some widely-used tokens like USDT do not return any value on transfer, which causes the boolean check to fail and revert even if the transfer was successful.

## Team Response

Fixed.
