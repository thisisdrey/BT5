# Usage of ERC777 token can block liquidation

## Summary
This bug report states that there is a potential issue with the use of ERC777 tokens in a specific contract. This could allow a malicious user to block liquidation, which could have negative consequences. The recommendation is to avoid using ERC777 tokens in this particular contract.

## Details
##### Description
Malicious user can use ERC777 token callbacks for blocking liquidation:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/credit/CreditManager.sol#L433
##### Recommendation
We recommend not to use ERC777 token as underlying token.
