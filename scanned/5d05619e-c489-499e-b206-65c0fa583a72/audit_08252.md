# WORLD WRITABLE TRANSLATIONS GOOGLE DOC

## Summary
The application has a bug where translations for different messages in different languages are fetched from a publicly editable Google Doc. The URL for this document is easily accessible and can be found in the installed application's files. This means that anyone can make changes to the document and when the developer runs a specific command, those changes will be imported into the application. This can potentially lead to malicious changes being made to the translations. The severity of this bug is rated as 4 out of 5 for both impact and likelihood. A video demonstrating this issue is also provided.

## Details
##### Description

The application fetches translations for various messages in various languages from a Google Doc which is world-writable.
The Google Doc URL is in plaintext and can be retrieved from the `app.asar` file of the installed application.
A malicious actor can edit this document to anything they want and the next time developer runs `npm run locale import all` they will get the malicious changes.
