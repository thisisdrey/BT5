# Unchecked delegatecall result

## Summary
This bug report describes a medium severity issue that has been resolved. In the contract Diamond.yul, there is a missing check for a delegatecall result in the constructor at line 30. This could potentially cause problems and it is recommended to add a check for the delegatecall result.

## Details
**Severity**: Medium

**Status**: Resolved

**Description**

In contract Diamond.yul at line 30, inside the constructor, there is a delegatecall performed back to the caller contract (AdminFacet) which executes the initializeFacet. The result of the delegate call is not checked for success/failure.

**Recommendation**: 

Add a check for the delegatecall result
