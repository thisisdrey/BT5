# Incorrect documentation (or parameter)

## Summary
The bug report is about a discrepancy between the documentation and the code in a file called PontoonTokenVesting.sol. The report states that on lines 126 and 156 in the constructor function, there is a discrepancy between the amount of supply mentioned in the documentation comment and the amount passed to the _addTeam() function. The same issue is present in the Strategic round, where the amount mentioned in the comment does not match the amount passed to the method. The recommendation is to verify the functionality and correct the comment or the documentation. The functionality has been verified, but the comment is still incorrect.
