# Lack of updating `_serviceCount` variable

## Summary
This bug report states that there is an issue with the addService() function in the FulfillableRegistry.sol file. The _serviceCount variable is not being updated, which causes the removeServiceAddress() function to always fail. The recommendation is to update the _serviceCount variable in the function. This bug has been resolved.

## Details
**Severity**: Medium

**Status**: Resolved

**Location**: FulfillableRegistry.sol

**Description**

The addService() function is missing to update the _serviceCount variable.
Due to a lack of increasing the variable, the removeServiceAddress() function will always fail.

**Recommendation**:

Update the _serviceCount variable in the function.
