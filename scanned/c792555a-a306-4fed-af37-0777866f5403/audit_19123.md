# User can approve any token

## Summary
The bug report states that there is an issue with a specific function in the Gearbox protocol. This function allows users to approve any token, which should not be possible. The report suggests adding a check to ensure that users can only approve certain tokens.

## Details
##### Description
User can approve any token:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/credit/CreditManager.sol#L761
##### Recommendation
We recommend adding a check that user can only approve allowable tokens.
