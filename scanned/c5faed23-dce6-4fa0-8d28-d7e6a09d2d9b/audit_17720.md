# [H-02] Wrong implementation of `withdrawAdminFees()` can cause the `adminFees` to be charged multiple times and therefore cause users' fund loss

## Summary
This bug report is about an issue in the SwapUtils.sol smart contract, which is part of the Connext project. It was submitted by WatchPug and concerns the `withdrawAdminFees()` function, which is used to transfer admin fees from the contract to an address. The issue is that the `self.adminFees[i]` is not being reset to 0 every time it is withdrawn, meaning that the admin fees can be withdrawn multiple times. This could be done maliciously, or the admin may just be unaware of the issue and withdraw the fees multiple times.

The recommended mitigation steps are to change the code to reset the `self.adminFees[i]` to 0 every time it is withdrawn. LayneHaber (Connext) confirmed and resolved this issue in the connext/nxtp@8eef974 commit, and 0xleastwood (judge) commented that this is a valid finding, as it could lead to LPs losing considerable value if the admin fees are withdrawn multiple times.

## Details
_Submitted by WatchPug_

[SwapUtils.sol#L1053-L1062](https://github.com/code-423n4/2022-06-connext/blob/b4532655071566b33c41eac46e75be29b4a381ed/contracts/contracts/core/connext/libraries/SwapUtils.sol#L1053-L1062)<br>

```solidity
function withdrawAdminFees(Swap storage self, address to) internal {
  IERC20[] memory pooledTokens = self.pooledTokens;
  for (uint256 i = 0; i < pooledTokens.length; i++) {
    IERC20 token = pooledTokens[i];
    uint256 balance = self.adminFees[i];
    if (balance != 0) {
      token.safeTransfer(to, balance);
    }
  }
}
```

`self.adminFees[i]` should be reset to 0 every time it's withdrawn. Otherwise, the `adminFees` can be withdrawn multiple times.

The admin may just be unaware of this issue and casualty `withdrawAdminFees()` from time to time, and rug all the users slowly.

### Recommended Mitigation Steps

Change to:

```solidity
function withdrawAdminFees(Swap storage self, address to) internal {
  IERC20[] memory pooledTokens = self.pooledTokens;
  for (uint256 i = 0; i < pooledTokens.length; i++) {
    IERC20 token = pooledTokens[i];
    uint256 balance = self.adminFees[i];
    if (balance != 0) {
      self.adminFees[i] = 0;
      token.safeTransfer(to, balance);
    }
  }
}
```
