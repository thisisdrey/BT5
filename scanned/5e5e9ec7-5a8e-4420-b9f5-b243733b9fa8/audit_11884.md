# `_feePercentage` Is Not Regulated

## Summary
The client has marked a bug in the code as "Fixed". The bug affects the file `UnikuraMothership.sol` and involves the variable `_feePercentage` not being properly regulated in the function `setFeePercentage()`. This variable is supposed to represent the percentage of the mint price that is required as a sales fee, but its range is not defined and it is not explained in the documentation. It is recommended to limit the value of `_feePercentage` to a specific range (such as 0 to 100) and to provide clear explanation of its purpose in the user-facing documentation.
