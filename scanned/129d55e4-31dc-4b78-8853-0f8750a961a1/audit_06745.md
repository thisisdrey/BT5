# M-6: incorrect condition for the iterative update of Equation 34

## Summary
The issue M-6 in the Allora Judging project was reported by user volodya. The issue is related to an incorrect condition in the code for the iterative update of Equation 34. According to the whitepaper, the iterative process should continue until the value is less than 0.001. However, in the code, it will stop at 0.001 times the maximum gradient threshold. This can lead to incorrect computation for listening coefficients. The code snippet and tool used for the review are not mentioned. The recommendation is to change the condition in the code to ensure the correct computation. The issue has been fixed by the protocol team in a recent pull request.
