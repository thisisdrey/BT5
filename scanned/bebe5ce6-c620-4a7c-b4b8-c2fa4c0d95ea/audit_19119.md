# Index not checked

## Summary
The bug report is about an issue with the code in the CurveV1.sol file of the Gearbox protocol contracts. The code is currently not checking if the index `i` is less than the value of `N_COINS`, which could potentially cause problems. The recommendation is to add a check to ensure that `i` is always less than `N_COINS`.

## Details
##### Description
Index `i` must be less than `N_COINS`:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/CurveV1.sol#L40
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/adapters/CurveV1.sol#L58
##### Recommendation
We recommend adding a check that `i < N_COINS`.
