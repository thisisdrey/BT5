# A broken hook can block user funds

## Summary
This bug report is about an issue related to closing positions in a contract. The problem is that when a user calls the `burn` function to close a position, it uses hooks to an external contract. If the hook were to break, then users would not be able to get their deposited funds.

The recommendation is to add an emergency exit for users that can be triggered if the hook breaks. This will ensure that users can still get their funds back in the event that the hook breaks.
