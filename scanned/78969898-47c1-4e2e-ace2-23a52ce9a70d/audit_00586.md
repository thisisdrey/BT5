# Pin Third Party Github Actions to Shasum

## Summary
The bug report is advising users to be careful when using external actions in their GitHub workflows. It is important to make sure that the version of the action remains the same to prevent failures or attacks. The recommendation is to reference the action using its SHA hash instead of just the version tag.

## Details
**Description:** When using external actions in your workflow, it is important to ensure that the version remains the same to avoid run-failures or supply chain attacks.

**Recommendation:** For every `uses` instruction in your GitHub actions, ensure that it is referenced by its SHA hash, and not just the version tag.
