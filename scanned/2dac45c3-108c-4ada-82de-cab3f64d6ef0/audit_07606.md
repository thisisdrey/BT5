# Inability to use NFT Marketplaces 

## Summary
This bug report is regarding a potential risk for users who are selling their NFTs (non-fungible tokens) through an online marketplace called MEME404. The issue is that if a user sends different NFTs to the marketplace, it could result in changes to the marketplace's tier, causing the loss of their NFT. The recommendation is for the marketplace to warn users about this risk and provide documentation about it. However, the marketplace has acknowledged this issue but will not fix it.

## Description
Users should be aware of the risks involved when selling their NFTs through an NFT marketplace. Potential losses could occur after creating an order in the marketplace. For instance, if users send different NFTs of MEME404 to the marketplace, it could result in changes to the marketplace tier through the burning/minting process. This could lead to the loss of their NFT.

## Recommendation
The recommendation is to warn users about the risks of selling on the NFT marketplace by providing documentation about it.

## Responses
- **Trugly Labs:** Acknowledged and won't fix.
