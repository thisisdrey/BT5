# Missing Gas Claim Functionality

## Summary
The current implementation of BazaarManagerBlast does not have a method for claiming gas. The constructor calls configureClaimableGas on the BLAST instance, but the contract lacks a dedicated method for actually claiming the gas. According to the Blast documentation, the default governor is the contract itself. Therefore, the contract should implement a method with the onlyOwner modifier to claim the gas. The solution is to introduce a method with the onlyOwner modifier for access control. This issue has been resolved in version 337b1a9 by Otter Audits LLC.

## Details
## BazaarManagerBlast Gas Claiming Issue

The current implementation of `BazaarManagerBlast` does not explicitly include a method for claiming gas. The constructor calls `configureClaimableGas` on the `BLAST` instance; however, the contract lacks a dedicated method to actually claim the gas. As per the Blast documentation, the default governor is the contract itself. Thus, the contract should implement a method with the `onlyOwner` modifier to claim the gas.

## Remediation

Introduce a method to claim the gas with the `onlyOwner` modifier for access control.

## Patch

Resolved in `337b1a9`.
