# General code quality

## Summary
This bug report focuses on the code quality of the DSTContract. It is noted that the code quality is low, making it difficult to audit. To improve the code quality, the report recommends a refactor. This will make the functionality of the code easier to understand and reduce the attack surface. Additionally, as the code is highly tested, any changes made to improve the code quality will have a very low risk of introducing any regressions.

## Details
Code quality of the DSTContract is low, which made auditing it hard. We recommend a refactor to improve code quality (more recommendations given next). Simpler code makes functionality more apparent and reduces attack surface. Given the high degree of test coverage, making changes to improve code quality will have a very low risk of introducing regressions.
