# M-20: Wrong calculation of `balanceBefore` and `balanceAfter` in deposit method

## Summary
This bug report is about an issue with the deposit method in the MainVault contract. The issue is that the calculation of the `balanceBefore` and `balanceAfter` is wrong when the `reservedFunds` is greater than the `getVaultBalance()`. This can lead to deposits failing when they should not. This issue was found by Ruhum, chaduke, and spyrosonic10. 

The code snippet that is causing the issue is:
```solidity
    uint256 balanceBefore = getVaultBalance() - reservedFunds;
    vaultCurrency.safeTransferFrom(msg.sender, address(this), _amount);
    uint256 balanceAfter = getVaultBalance() - reservedFunds;
    uint256 amount = balanceAfter - balanceBefore;
```

The tool used to find this issue was manual review. The recommendation is to use the following code instead:
```solidity
    uint256 balanceBefore = getVaultBalance();
    vaultCurrency.safeTransferFrom(msg.sender, address(this), _amount);
    uint256 balanceAfter = getVaultBalance();
    uint256 amount = balanceAfter - balanceBefore;
```

Finally, the discussion section mentions that this issue is a duplicate of three other issues (#262, #66, #45).

## Summary
Deposit method calculate net amount transferred from user.  It use `reservedFunds` also in consideration when calculating `balanceBefore` and `balanceAfter` but it is not actually require.

```solidity
    uint256 balanceBefore = getVaultBalance() - reservedFunds;
    vaultCurrency.safeTransferFrom(msg.sender, address(this), _amount);
    uint256 balanceAfter = getVaultBalance() - reservedFunds;
    uint256 amount = balanceAfter - balanceBefore;
```

## Vulnerability Detail
Deposit may fail when `reservedFunds` is greater than `getVaultBalance()`

## Impact

Manual Review

## Recommendation
Use below code. This is correct way of finding net amount transfer by depositor
```solidity
    uint256 balanceBefore = getVaultBalance();
    vaultCurrency.safeTransferFrom(msg.sender, address(this), _amount);
    uint256 balanceAfter = getVaultBalance();
    uint256 amount = balanceAfter - balanceBefore;
```
