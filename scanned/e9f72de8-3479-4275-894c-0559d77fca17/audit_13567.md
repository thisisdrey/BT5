# A manager could transfer funds to the Claiming contract instead of the athlete

## Summary
This bug report is about a manager being able to transfer funds to the Claiming contract instead of the athlete. The link provided in the report leads to a code snippet of a contract called FantiumClaimingV1.sol. The bug report states that the manager should not be using `_msgSender()` as this will cause the funds to be sent to the Claiming contract instead of the athlete. The recommendation is to use the athlete's address instead of `_msgSender()`.

## Details
##### Description
A manager could transfer funds to the Claiming contract instead of the athlete here https://github.com/FantiumAG/smart-contracts/blob/c622b4c35132167bacc22cf76656d80715edb5fd/contracts/claiming/FantiumClaimingV1.sol#L303.

##### Recommendation
We recommend using the athlete's address instead of `_msgSender()`.
