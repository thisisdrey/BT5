# Addresses not checked

## Summary
This bug report is about the code in the StakingRewardsV3-1.sol contract, which is part of the Keep3r Network. The bug is that the code does not check if the input address is equal to zero address. This can lead to some unexpected results when using the contract.

The recommendation is to add a check to the code, so that it is ensured that the input address is not equal to zero address. This will help to make sure that the contract works as expected.
