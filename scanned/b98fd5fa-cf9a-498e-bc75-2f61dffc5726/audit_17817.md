# Unit-tests fail due to changes.

## Summary
This bug report discusses an issue with a function called addGauge() in the GaugeProxy.sol file. The bug occurs when the function is called from three different files, GaugeProxyBribeMiddleManTesting.js, GaugeProxyDistributeTesting.js, and GaugeProxyBribeTesting.js. In all three cases, the function fails due to incorrect parameters being passed to it. The recommendation is to pass the correct parameters to the function call.

## Details
**Description**

GaugeProxyBribeMiddleManTesting.js: before() hook.
GaugeProxyDistributeTesting.js: before() hook, it('Admin functions only, all tests to revert) test.
GaugeProxyBribeTesting.js: before() hook, it('Add a new Gauge for LP4') test.
Call to function addGauge() from GaugeProxy.sol fails due to change of parameters.

**Recommendation**:

Pass the correct parameters to function call.
