# Unlimited approval

## Summary
This bug report is about a line of code located at https://github.com/CoverProtocol/cover-peripheral/tree/d5b37e34d47abec3252cdabd46e55e34a72421d4/contracts/Rollover.sol#L64. This line of code, `_token.approve(_spender, uint256(-1)`, gives CoverRouter an unlimited approval access. This could be a security risk, so it is recommended to either revoke the approval after the execution or to approve only the expected amount.

## Details
##### Description
At this line `_token.approve(_spender, uint256(-1)` located at https://github.com/CoverProtocol/cover-peripheral/tree/d5b37e34d47abec3252cdabd46e55e34a72421d4/contracts/Rollover.sol#L64 we have an unlimited approval access from CoverRouter.

##### Recommendation
We recommend either to revoke the approval after the execution or to approve only the expected amount.
