# No cap for Minter

## Summary
A bug report is an important document that outlines a problem that needs to be fixed. In this bug report, it is noted that there is no cap for Minter, which could lead to the Minter quickly reaching the maximum supply. To solve this issue, it is recommended that a minterLimit be added for Minter roles. This minterLimit would restrict the amount that is allowed to be minted by the Minter role. This would help to prevent the Minter from quickly reaching the maximum supply.

## Details
**Description:** 

Since there is no cap for Minter,he could end up reaching `maxSupply` quickly.

**Recommendations:** 

Add a `minterLimit`for Minter roles which will restrict the amount which is allowed to be minted by minter role.
