# Unable to compile contracts ✓ Fixed

## Summary
This bug report is about a compilation issue with two repositories, `Fairswap_iDOLvsImmortalOptions` and `Fairswap_iDOLvsLien`. In the first repository, the compilation with truffle failed due to a missing file (`contracts/testTokens/TestBondMaker.sol`) and with `solc` due to an undefined interface function. In the second repository, the compilation with truffle failed due to a missing file (`./ERC20RegularlyRecord.sol`) with the correct filename being `./TestERC20RegularlyRecord.sol`.

The related code was updated on the 7th day of the audit, and the issue was fixed as far as reviewed. The recommendation is to ensure all contracts are easily compilable by following the instructions in the README.
