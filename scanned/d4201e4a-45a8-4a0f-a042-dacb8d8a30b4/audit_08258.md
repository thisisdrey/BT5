# HARDCODED PRIVATE KEYS STORED ON DISK

## Summary
This bug report is about a security issue in the Anatha Desktop Wallet. The report states that private keys for wallets belonging to the Anatha Organization are being leaked. Private keys are important for securing wallets and if they are leaked, attackers can use them to make transactions on those wallets. The report identifies five locations in the code where these private keys can be found and recommends that they be removed before the wallet is deployed. The report also includes two images showing the leakage of private keys in the app.asar file. The impact of this issue is rated as 5 out of 10 and the likelihood of it happening is also rated as 5 out of 10.

## Details
##### Description

During the assessment, we have identified that the Anatha Desktop Wallet leaks private keys for wallets that belong to the Anatha Organization. The private keys in the `[Installation Folder]/resources/app.asar` file. Any attacker who installs the Anatha Desktop Wallet can retrieve the keys and use them to perform transaction on the respective wallets. There are five such leaked private keys.
