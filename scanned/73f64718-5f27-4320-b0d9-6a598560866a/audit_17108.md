# Unchecked possible zero address for `_to` parameter in EthBridge.

## Summary
The report is about a bug in a contract called EthBridge.sol. The bug is in line 69 of the contract's function called depositERC20To, where the _to function parameter can be set to zero and is not checked. This can lead to problems such as not being able to receive deposited tokens. The recommendation is to add a check to make sure the _to address is not set to zero. This issue has been resolved.
