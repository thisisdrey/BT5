# Extra parameter for the method

## Summary
This bug report is about the PolygonBridgeExecutor.sol contract file. At line 35 of this file, the processMessageFromRoot() method is called with a parameter called stateId. However, this parameter is not used anywhere in the method, and it is recommended to remove it. This bug report highlights the importance of thoroughly checking code for any unused parameters or variables. Removing these can help reduce the size of the code, and make it easier to debug and maintain.

## Details
##### Description
At the line https://github.com/aave/governance-crosschain-bridges/blob/9fd0609a2e14d546885f76211961f251d2e15cb9/contracts/PolygonBridgeExecutor.sol#L35, when calling the `processMessageFromRoot()` method, the `stateId` parameter is passed.
But then this variable is not used anywhere in the method.

##### Recommendation
It is recommended to remove the unused parameter.
