# M-3: The default value of epsilon differs from what is stated in the whitepaper

## Summary
The reported bug is related to the default value of epsilon in the protocol, which is different from what is stated in the whitepaper. This was found by a user named "volodya" and has been acknowledged by the protocol. The impact of this bug is that the precision will be tighter. The recommendation is to change the value of epsilon from 0.0001 to 0.01. The bug was found through a manual review and one comment has been left on the issue during the judging contest.
