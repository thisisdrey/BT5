# Code is not production-ready ✓ Fixed

## Summary
This bug report is about the code in the file AuctionTimeControl.sol of the MainContracts. This code contains several functions that have been commented out and replaced with functions that simply return True for testing purposes. These commented-out functions are essential for the Auction contract to work properly, such as the function inRevealingValuationPeriod which is used to allow revealing of the bid price publicly. The report recommends removing the test functions and using production code for testing, as the tests must have full coverage of the production code to be considered complete. The comment from Lien Protocol suggests to remove comment out lines and update code to activate time control functions at AuctionTimeControl.sol.
