# Function add() is not restricted.

## Summary
The bug report describes an issue with the function add() in the Storage.sol file. This function has public access and no restrictions, which means anyone can use it. However, this also means that anyone can push unlimited data to the storage, potentially causing the contract to become unusable. The recommendation is to either complete the logic for the function or remove the commented code. It is also mentioned that the Storage.sol file has been deleted after being audited.
