# Insufficient Access Controls on collectPositionSwapFee Will Allow Anyone to Call This Function

## Summary
This bug report is about a contract called TeaVaultAlgebraV1Point9. The report says that there is a problem with the function called collectPositionSwapFee. According to the contract's specifications, only the fund manager should be able to call this function. However, there is no restriction in place, which means that anyone can call it. The report suggests adding a modifier called onlyManager to the function to fix this issue. The bug has been resolved.

## Details
**Severity**: Medium

**Status**: Resolved

**Description**:

According to the nat spec for the TeaVaultAlgebraV1Point9 contract, the fund manager should only be allowed to call collectPositionSwapFee however, there appears to be no access controls placed on the function implementation which may allow anyone to call this function.

**Recommendation**:

Add the onlyManager modifier to the collectPositionSwapFee function for the TeaVaultV1Point9 contract.
