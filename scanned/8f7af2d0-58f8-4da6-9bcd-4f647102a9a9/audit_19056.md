# Use the function ‘_mint’ in the constructor

## Summary
The bug report states that there is an issue with line 18 in the code. It suggests that all changes to balances should be done through ERC20 functions. The report recommends using the function '_mint' to properly mint tokens for the sender. This function will update the mapping '_balances' and the variable '_totalSupply' as well as emit an event to show that tokens were minted.

## Details
**Description**

Line 18. All changes to balances should be performed through ERC20 functions.

**Recommendation**:

Use function ‘_mint’ in order to mint sender tokens properly. Function ‘_mint’ will write a given
value to mapping ‘_balances’ from ERC20, as well as update variable ‘_totalSupply’ and emit an
event that tokens were minted.
