# Owner can set unlimited amount

## Summary
This bug report describes a vulnerability in the Cumulative Merkle Drop contracts, which allows the owner of an account to set a merkle root with an unlimited amount of allowed funds. The vulnerability is present in the CumulativeMerkleDrop128, CumulativeMerkleDrop and CumulativeMerkleDrop160 contracts.

The recommendation to address this vulnerability is to use a multisig wallet for the owner of the account. This will ensure that any changes to the contract require multiple signatures, making it more difficult for malicious actors to exploit the vulnerability.
