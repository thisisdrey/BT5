# LARGE VALUES IN INPUT FIELDS SITE CRASH

## Summary
The report states that there is a bug on the website `https://hsuite-finance.firebaseapp.com` when large numbers are entered into the `amount` input fields. This causes the site to become unresponsive and requires a manual refresh of the browser. The issue is due to the numbers being converted into scientific notation. The impact and likelihood of the bug are both rated as 3 out of 5. The recommendation is to use `encodingUrl` on all parameters, which has been implemented by the `HbarSuite team` to solve the issue.

## Details
##### Description

It was found that `https://hsuite-finance.firebaseapp.com` when large values are entered into the `amount` input fields, an error is generated and the site becomes unresponsive until a manual refresh of the browser. It appeared that the error was due to the number being converted into scientific notation.
