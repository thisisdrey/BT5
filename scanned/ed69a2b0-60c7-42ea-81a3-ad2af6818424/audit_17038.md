# Unsound crank mechanism

## Summary
This report discusses a bug found in the GooseFX SSL Audit 04, specifically in the file "swap.rs" written in the programming language RUST. The bug is located in lines 91-93 where the function "apply" is defined. The issue is that the code is not properly assigning values to variables, which can lead to vulnerabilities in the SSL system. The bug has been fixed in the latest patches, specifically versions 69c0513, 303bd06, and 1f24518. This report was created by OtterSec LLC and all rights are reserved.
