# Tree Level incompatible

## Summary
The bug report is about an issue with two contracts, dpnm_sc.sol and phenomanelTree.sol, where the tree depth is not matching between them. The dpnmMain contract assumes a tree depth of 10 while the phenomanelTree contract is built for a depth of 15. This has been identified as a high severity issue and has been resolved by the developers. The recommendation is to match the tree depth between the two contracts. The fix is to leave room for future products by making the depth adjustable up to 15, according to the developers.

## Details
**Severity**: High

**Status**: Resolved

**Description**

dpnm_sc.sol / phenomanelTree.sol - tree depth is incompatible between the dpnmMain and phenomanelTree. Contract dpnmMain assumes depth of tree is 10 while phenomanelTree is built to be 15.

**Recommendation** 

Match the tree depth.

**Fix** - It is meant to be as a spec in order to leave room for future products as according to developers depth will be adjustable up to 15.
