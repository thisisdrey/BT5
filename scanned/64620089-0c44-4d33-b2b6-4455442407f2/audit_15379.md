# Use `transfer` instead of `call`

## Summary
This bug report is about the `transfer` method in a project. The report recommends using the `transfer` method as opposed to other contracts because it forwards 2300 gas, making it safer. The report also suggests that using the `transfer` method will help to avoid potential re-entrancy. In conclusion, the bug report recommends using the `transfer` method for increased safety and to avoid potential re-entrancy.

## Details
##### Description

If the project work with other contracts is not implied, then the `transfer` method is safer (this forwards 2300 gas).

##### Recommendation

We recommend using `transfer` to avoid potential re-entrancy.
