# M-1: CRITICAL: Access Control Bypass in Fee Management

## Summary
This bug report is about a vulnerability found in the MarketController.sol file of a project called "2025-10-index-fun-order-book-contest-judging". The vulnerability allows authorized users to manipulate system parameters, such as fees and trading pauses. This can lead to complete control of the system by malicious users. The team has acknowledged the issue but will not fix it at this time. The recommended fix is to change the modifier from "onlyAuthorizedMatcher" to "onlyOwner" in three functions. The vulnerability was confirmed through manual code review and automated static analysis.

## Description
Critical administrative functions use `onlyAuthorizedMatcher` instead of `onlyOwner`, allowing any authorized matcher to manipulate system parameters.

## Code Snippet
```solidity
function setFeeRate(uint256 _feeRate) external onlyAuthorizedMatcher { // WRONG MODIFIER
    require(_feeRate <= MAX_FEE_RATE, "Fee rate exceeds maximum");
    uint256 oldRate = feeRate;
    feeRate = _feeRate;
    emit FeeRateUpdated(oldRate, _feeRate);
}

function setTreasury(address _treasury) external onlyAuthorizedMatcher { // WRONG MODIFIER
    require(_treasury != address(0), "Invalid treasury address");
    address oldTreasury = treasury;
    treasury = _treasury;
    emit TreasuryUpdated(oldTreasury, _treasury);
}

function setGlobalTradingPaused(bool paused) external onlyAuthorizedMatcher { // WRONG MODIFIER
    globalTradingPaused = paused;
    emit GlobalTradingPauseChanged(paused);
}
```

## Impact
- Matchers can set fees up to 10% maximum
- Matchers can redirect treasury funds to any address
- Matchers can pause/resume all trading globally
- Complete system control by malicious matchers

## Recommended Fix
```solidity
function setFeeRate(uint256 _feeRate) external onlyOwner { // CORRECT MODIFIER
    require(_feeRate <= MAX_FEE_RATE, "Fee rate exceeds maximum");
    uint256 oldRate = feeRate;
    feeRate = _feeRate;
    emit FeeRateUpdated(oldRate, _feeRate);
}

function setTreasury(address _treasury) external onlyOwner { // CORRECT MODIFIER
    require(_treasury != address(0), "Invalid treasury address");
    address oldTreasury = treasury;
    treasury = _treasury;
    emit TreasuryUpdated(oldTreasury, _treasury);
}

function setGlobalTradingPaused(bool paused) external onlyOwner { // CORRECT MODIFIER
    globalTradingPaused = paused;
    emit GlobalTradingPauseChanged(paused);
}
```
