# Unexpected length of array returned

## Summary
The bug report is about a function called "scaleArray" in a file called "Solarray.sol". The severity of the bug is medium and it has been resolved. The function is supposed to return an array of 4 numbers, but instead it is returning an array of 5 numbers. The recommendation is to change the return statement to match the intended length of 4 numbers.

## Details
**Severity**: Medium

**Status**: Resolved

**Description**

Solarray.sol - In scaleArray - function scaleArray(uint256[4] memory _array, uint256 _scale) internal pure returns (uint256[5] memory _retArray) {

_retArray is of length 5 while it should be 4.

**Recommendation** 

Change return to (uint256[4] memory _retArray).
