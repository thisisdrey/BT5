# Restriction cannot be set again

## Summary
The bug report is about a feature in a code called contracts/RUSD.sol that is not working correctly. The feature is related to setting restrictions and updating them. The report suggests checking the functionality and verifying if the state changes are correct. It also recommends adding documentation and fixing the flow of restrictions change. The ReSource team has verified the functionality after the audit.
