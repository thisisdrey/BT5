# `EigenLSTRestaking` Strategy Manager May Arbitrarily Withdraw stEth

## Summary
The client has marked a bug as "Fixed" in the file `EigenLSTRestaking.sol`. The bug allows the strategy manager to make unauthorized calls on the `tokenAddr` contract using the `invoke()` function. This can be used to withdraw funds or claim queued withdrawals without permission. The recommendation is to remove the `invoke()` function and create separate functions for each use case.

## Details
**Update**
Marked as "Fixed" by the client. Addressed in: `9589aa88b5b2055c7e9df4563c3ecb6d623f637a`.

**File(s) affected:**`EigenLSTRestaking.sol`

**Description:** The strategy manager may perform arbitrary calls on the `tokenAddr` contract by calling `invoke()`. This could be abused to request withdrawals and/or claim any queued withdrawals to oneself.

**Recommendation:** We recommend removing this function and creating dedicated functions for each use case rather than an all-purpose `invoke()` function.
