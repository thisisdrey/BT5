# LAUNCHPAD PURCHASE PAIRING SWAPPED

## Summary
This report discusses a bug found on the website https://hsuite-finance.firebaseapp.com when purchasing $LPT tokens on the Launchpad. It was discovered that if a negative value is entered, the tokens are swapped and the user ends up spending $LPT instead of earning it. The impact of this bug is rated as a 3 out of 5 and the likelihood of it occurring is also rated as a 3 out of 5. The recommendation is that the HbarSuite team has implemented a fix to prevent negative values from being entered into the input fields.

## Details
##### Description

It was found that when a negative value is provided when purchasing `$LPT` in `Launchpad` on `https://hsuite-finance.firebaseapp.com`, the tokens are swapped around and you will be spending `$LPT` instead of earning it.

**SOLVED**: The `HbarSuite team` implemented a fix by making sure no negative values can be entered into the input fields.
