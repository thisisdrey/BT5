# The wrong stake amount is stored.

## Summary
The bug report is about a function called "function_deposit()" in a file called "TokensFarm.sol". The issue is that there is a mistake in the code where the amount being stored is incorrect when there is a stake fee involved. The recommendation is to fix this by storing the correct amount in a variable called "stake.amount". This issue has been resolved.

## Details
**Description**

Perpetual TokensFarm.sol: function_deposit(), line 1083.
The '_amount parameter is stored in 'stake.amount . In case there is a stake fee, a greater amount would be stored instead of the value after taking the fee.

**Recommendation**

Store 'staked Amount in 'stake.amount`.

Resolved
