# H-7: traceurl - miss updating pool's token balance after finishLiquidation()

## Summary
This bug report is about an issue identified in the D3Liquidation's finishLiquidation() function where the updating of a pool's token balance is missing. This was found by the Protocol Team and tracerul during a manual review. The impact of this issue is that there will be an unequal record of pool's token balance and IERC20(token).balanceOf(pool), which can cause serious problems. The recommendation to fix this issue is to call updateReserveByVault after pool transfer token to vault.
