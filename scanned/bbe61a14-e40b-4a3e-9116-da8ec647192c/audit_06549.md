# Implement TWAP for More Accurate Price Retrieval in Price Oracle

## Summary
This bug report discusses a problem with the setPrice function in the Entangle Protocol. The issue was caused by the function not properly calculating the TWAP (Time-Weighted Average Price) and instead using a static price. The team recommends modifying the function to accurately calculate the TWAP by fetching multiple price points within a specific time frame and averaging them. The issue has been resolved by the team by calculating the price with the synth amount. The link to the code fix is provided at the end of the report.
