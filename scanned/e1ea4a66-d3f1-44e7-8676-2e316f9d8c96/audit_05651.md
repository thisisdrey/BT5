# [M-01] Users can evade paying fees

## Summary
This bug report is about a problem in the `TokenFactory` code. The report states that there is a bug where users can avoid paying fees when buying and selling tokens. This is because the code allows users to trade tokens in the open market without paying fees. The report recommends that the code should be changed to pause all transfers when `curve.isPaused` is `false`.

## Details
## Severity

**Impact:** Medium

**Likelihood:** Medium

## Description

The `TokenFactory` charges fees for both buying and selling tokens.

```solidity
        // initialize fees
        protocolFeeBips = 150;
        creatorFeeBips = 250;

        protocolFeeWithReferralBips = 100;
        creatorFeeWithReferralBips = 250;
        referralFeeBips = 50;
```

The issue here is that users can evade paying both fees by simply trading their tokens in the open market (e.g. Dex, Uniswap pairs, etc...)

## Recommendations

Consider pausing all transfers when `curve.isPaused` is `false`.
