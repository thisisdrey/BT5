# There is no check of the ether balance on the contract before it is transferred

## Summary
This bug report is about the line of code found at https://github.com/iearn-finance/yoracle.link/blob/faf1309cbe7a05f70b338351315039eb8e5b9c09/contracts/Keep3rV1Oracle.sol#L479. This line of code is responsible for transferring ether from the contract. The issue is that the code does not check that the amount of ether is greater than 0 before transferring it.

The recommendation is to add a check to make sure the ether balance on the contract is greater than 0 before transferring it. This will ensure that the code does not transfer ether if there is not enough of it in the contract.
