# [M-14] Owner can steal Concur rewards

## Summary
This bug report is about a vulnerability in the MasterChef.sol contract. Through this vulnerability, the owner of the contract can steal rewards from the Concur by adding a depositor and inflating other depositors' assigned balance of the token within the contract. This would allow the owner-managed depositor to get most (all but one wei) of the created tokens. The vulnerability was discovered through manual analysis. To mitigate this vulnerability, it is recommended that the owner should not be allowed to add depositors after the depositors have been configured.
