# USE OF PACKAGES WITH KNOWN VULNERABILITIES

## Summary
This bug report is about a vulnerability found in the Anatha Desktop Wallet application. The app uses a third-party dependency to handle program arguments, but this dependency has some high and critical vulnerabilities that can affect the security of the app. The affected packages are `axios`, `elliptic`, `follow-redirects`, `lodash`, and `node-notifier`. The report also includes the location of the code and the output of the `npm audit` command, which shows the details of the vulnerabilities and the suggested fix. The report also lists the dependencies and their versions, which are affected by the vulnerabilities. The score for this vulnerability is 4 for impact and 3 for likelihood.

## Details
##### Description

The Anatha Desktop Wallet application uses a third-party dependency to delegate handling of program arguments. However, the dependency creates an expected disadvantage in that the actual application's security posture now rests on it.

Specifically, the `axios <=0.21.1` package suffers from a high vulnerability, the `elliptic<=6.5.3` package suffers from a high vulnerability, the `follow-redirects<=1.14.7` suffers from a high vulnerability, the `lodash<=4.17.20` package suffers from a critical vulnerability and the `node-notifier<8.0.1` suffers from a medium vulnerability.
