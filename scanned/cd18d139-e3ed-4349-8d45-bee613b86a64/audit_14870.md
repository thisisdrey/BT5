# For loop over dynamic array

## Summary
This bug report states that there is a problem with three functions called _updateRewards(), recoverClaim(), and claim(). These functions have a for loop over a dynamic array, which means the size of the array can change. However, if the array becomes too large, it can cause issues with running out of gas. To fix this problem, it is recommended to add a check on the maximum length that the array can be.

## Details
**Severity**: Medium

**Status**: Acknowledged

**Description**

The functions _updateRewards(), recoverClaim() and claim() function have a for loop over a dynamic array. This can lead to out of gas issues if the array becomes too large.

**Recommendation**: 

It is advised to add a check on the max length that the array can take.
