# NFTR-2 | Hold Farming Naming Not Free

## Summary
This bug report concerns the NFT (non-fungible token) function, `changeName`. According to the documentation, if an NFT is curated and is in the hold farming period, naming it should be free. However, the `changeName` function still asks for payment in the specified currency even if the NFT collection is curated and in the hold farming period.

The recommendation is to implement logic such that a NFT in the holding period can be named for free. The NFTR Team has implemented the suggested changes, resolving the issue.
