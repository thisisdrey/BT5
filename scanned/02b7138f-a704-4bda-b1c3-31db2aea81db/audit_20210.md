# collectExitFees, collectPerformanceFees don't check transfer result

## Summary
This bug report is related to the `freeTokens` value in the Liquidity Vault. It has been identified that `freeTokens` value will be decreased even if token transfer fails. This occurs at lines 478, 483, 516, and 521 of the Liquidity Vault. To fix this issue, it is recommended to use `safeTransfer` instead of `transfer`.
