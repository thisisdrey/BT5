# Profit distribution can be reset by owner at any time

## Summary
This bug report is about the function [`PrepareForProfitShare`](https://github.com/fiinu/smart-contract/blob/7589b9d137bbf902e142e4f0476b06976e390bdb/FiinuCoin.sol#L277) which is callable by the owner at any time and resets the distribution of profits, reducing every token holder's claimable profits to zero. The purpose of this function is not clear, and it harms the trustlessness of the contract. Therefore, it is suggested that the function is removed.

## Details
There is a function [`PrepareForProfitShare`](https://github.com/fiinu/smart-contract/blob/7589b9d137bbf902e142e4f0476b06976e390bdb/FiinuCoin.sol#L277) callable by the owner at any time, which resets the distribution of profits: it brings every token holder’s claimable profits down to zero. It’s not clear what purpose it serves. Consider removing it, as it harms the trustlessness of the contract.
