# withdrawRevenue() transfers USDB back to itself 

## Summary
The bug report is about a function called withdrawRevenue() in the QuailFinance.sol code. This function is transferring a type of currency called USDB back to the contract, which is unnecessary because the funds are already in the contract. The function was meant to withdraw the funds to the owner's address or a specified address, but it is not currently doing so. The recommendation is to either withdraw the funds to the owner's address or add an argument to specify a different address to transfer the funds to.
