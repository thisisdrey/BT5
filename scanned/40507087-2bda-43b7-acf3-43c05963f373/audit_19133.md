# Length of input arrays not checked

## Summary
The bug report is about a missing check in a code that can cause issues. The code is used for credit filtering and the problem is that the length of input arrays is not being checked. This means that if the arrays have different lengths, it can cause errors or unexpected behavior. The recommendation is to add a check to ensure that the lengths of the arrays are equal to prevent any potential issues.

## Details
##### Description
Length of input arrays is not checked:
https://github.com/Gearbox-protocol/gearbox-contracts/blob/0ac33ba87212ce056ac6b6357ad74161d417158a/contracts/credit/CreditFilter.sol#L297
##### Recommendation
We recommend adding a check that lengths of arrays are equal.
