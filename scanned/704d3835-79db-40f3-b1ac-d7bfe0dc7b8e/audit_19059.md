# Minting tokens is not restricted.

## Summary
The bug report is about a function called "mint()" on line 30 that allows anyone to create an unlimited number of tokens. The recommendation is to add restrictions to this function, such as only allowing the owner to use it. This will prevent the creation of an excessive amount of tokens.

## Details
**Description**

Line 30, function mint(). Currently, anyone can call this function, thus minting as many tokens
as one wants.

**Recommendation**:

Consider adding restrictions to the mint() function(such as onlyOwner).
