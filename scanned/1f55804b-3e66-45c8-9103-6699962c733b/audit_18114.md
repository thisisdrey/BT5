# [M-05] ABDKMath64 performs multiplication on results of division

## Summary
This bug report is about a vulnerability in the Solidity code in the ABDKMath64x64.sol library. The vulnerability is that Solidity could truncate the results of calculations, leading to incorrect results. To mitigate this, it is recommended that the code be changed to perform multiplication first, before any division is done. This will prevent any rounding or truncation from taking place.
