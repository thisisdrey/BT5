# APR Can Be Higher Than Intended

## Summary
This bug report is about an issue found during the testing phase of a program called EsEMBR.sol. The problem was with the payOffWeeks function, specifically with the APR calculation. The issue occurred when trying to pay off multiple weeks but not having enough funds. The client fixed the problem by adding code to only increase the debt if there was enough to pay off the following week. The bug has been resolved.
