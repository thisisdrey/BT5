# [H-01] Broken `NonceVoter` Allows Observer to Halt the Chain

## Summary
The `NonceVoter` function in the ZetaChain code is not functioning correctly. It does not count votes or perform any other operations as it is supposed to. This can be exploited by a single observer to change the nonces of connected chains, which can lead to the chain not functioning properly and causing loss of funds. The code that is supposed to handle voting is not working as intended, and instead, the chain nonce is simply set without any checks. This issue can be tested in the smoke-tests.

## Details
As it is, `NonceVoter` does not count votes or perform any similar operations. A single observer can use it to change chain nonces. Although there is a comment pointing this function is deprecated, it critically influences the system with its broken implementation. Arbitrarily changing nonces for the connected chains can be used to completely break the functioning of the chain and cause loss of funds.
