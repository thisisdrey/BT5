# Wrong daily rate value

## Summary
The bug report states that there is an issue with the daily rate value in a specific section of code. The code currently shows a daily rate of 8.33%, but according to the whitepaper, it should be 5.55%. The recommendation is to change the daily rate value to match the whitepaper.

## Details
**Description**

// 5: Team 7%, 7,000,000, 6 Month LOCK, 12 months @ rate of 8.33% per month
vestingTypes.push(VestingType(277777777777777800, 180 days, false));
On lines 67 and 68 daily rate value is 8.33% but in the whitepaper, it is 5.55%.

**Recommendation**:

Change the value of the daily rate.
