# Function only returns 'true'.

## Summary
The bug report discusses a problem with two functions in the code, one in ReferralRegistry.sol and the other in FNDZController.sol. The function __isReferredAddress() is not returning values in all cases, instead it only returns 'true' every time. The recommendation is to fix the function so that it returns values every time it is called.

## Details
**Description**

ReferralRegistry.sol: function __isReferredAddress().
FNDZController.sol: isDenominationAssetApproved().
Function returns only 'true' value and not in all cases. Function should return values every
time.

**Recommendation**:

Make sure to return value every time.
