# Incorrect slippage protection on deposits

## Summary
This bug report is about incorrect slippage protection on deposits. The slippage on deposits is enforced by the `minMarketTokenAmt` parameter but in the calculation of `minMarketTokenAmt`, the slippage is factored on the user's deposit value and not the leveraged amount which is actually being deposited to GMX. This can lead to depositors losing value.

For example, if the vault is a 3x leveraged vault and a user deposits 1 usd worth tokenA and sets slippage to 1%, the `minMarketTokenAmt` calculated is worth 0.99 usd but the actual deposit added is worth 3 usd due to leverage, thus the deposit receives 2.90 worth of LP token which is more than 1% slippage.

The recommendation to fix this issue is to use the actual deposit value instead of the user's initial deposit value when calculating the `minMarketTokenAmt`. The relevant GitHub link is provided in the bug report.

## Vulnerability Details
The slippage on deposits is enforced by the `minMarketTokenAmt` parameter. But in the calculation of `minMarketTokenAmt`, the slippage is factored on the user's deposit value and not the leveraged amount which is actually being deposited to GMX.
```solidity

    GMXTypes.Store storage self,
    GMXTypes.DepositParams memory dp,
    bool isNative
  ) external {
    
    ....... more code

      // If LP token deposited

        address(self.lpToken),

        address(self.tokenA),
        address(self.tokenB),
        false,
        false

      * dp.amt

      // If tokenA or tokenB deposited
      _dc.depositValue = GMXReader.convertToUsdValue(
        self,
        address(dp.token),
        dp.amt
      );
    }

      self,
      _dc.depositValue,
```

But vaults with leverage greater than 1 will be adding more than `_dc.depositValue` worth of liquidity in which case the calculated `minMarketTokenAmt` will result in a much higher slippage.

### Example Scenario
- The vault is a 3x leveraged vault
- User deposits 1 usd worth tokenA and sets slippage to 1%.
- The `minMarketTokenAmt` calculated is worth 0.99 usd
- The actual deposit added is worth 3 usd due to leverage
- The deposit receives 2.90 worth of LP token which is more than 1% slippage

## Impact
Depositors can loose value

## Recommendations
Use the actual deposit value instead of the user's initial deposit value when calculating the `minMarketTokenAmt`
