# Lack of implementation to remove the whitelisted address

## Summary
A bug report has been filed regarding the Shop.sol contract. This contract uses a mapping (uint256 => mapping(address => bool)) storeIdToUserInWhitelist to store whitelisted addresses for each store ID. The whitelisting process is done by onlyStoreManagers with a specific function. These whitelisted addresses are then allowed to interact with the function buyProduct, even when the shop is closed. However, it has been found that this process is not working correctly, and the whitelisted addresses are not being allowed to buy products. This bug needs to be fixed to ensure that the whitelisted addresses are able to buy products when the shop is closed.
