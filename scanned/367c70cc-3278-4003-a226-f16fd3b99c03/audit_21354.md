# Interface Consistency

## Summary
This bug report is about the need to update the StrategyBase interface in order to keep it consistent. The StrategyCurveSCRVv4_1 interface was changed with recent audit results, and it is recommended to update all other StrategyBase-interfaced strategies in order to maintain consistency. The getMostPremium() and harvest() functions must be updated to keep the interface compliant and avoid any issues with asset index results or front-running. This has now been fixed with the release of https://github.com/pickle-finance/protocol/pull/19.

## Details
This warning is about the `StrategyBase` interface consistency.
