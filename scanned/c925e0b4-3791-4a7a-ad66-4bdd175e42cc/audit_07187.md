# Uncontrolled Resource Consumption in go-jose (github.com/dvsekhvalnov/jose2go)

## Summary
This report describes a medium severity bug in a library called github.com/dvsekhvalnov/jose2go. The bug allows an attacker to cause excessive memory or CPU usage, potentially leading to a denial-of-service attack. However, the likelihood of this happening is low as it requires the attacker to send specific requests. To fix this issue, it is recommended to upgrade to the latest version of the library.

## Details
**Severity:** Medium

**Status:** Acknowledged

**Description:** 

The library is vulnerable to uncontrolled resource consumption due to improper input validation, which could lead to excessive memory or CPU usage.

**Impact:** 

An attacker could exploit this vulnerability to cause a denial-of-service (DoS) attack.

**Likelihood:** 

Low, as it requires the ability to send specially crafted requests to the application.

**Recommendation:** 

Upgrade github.com/dvsekhvalnov/jose2go to the latest version to mitigate this issue.
