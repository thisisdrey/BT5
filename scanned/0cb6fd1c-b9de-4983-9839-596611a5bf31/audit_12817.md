# gauges Mapping Cannot Be Cleared

## Summary
The bug report discusses an issue with the `killGaugeTotally` function in the VoterV3.sol contract. This function deletes the `_gauge` key in the `poolForGauge` mapping and then later retrieves it to clear the `gauges` mapping. However, this retrieval always returns the zero address, causing the pool address to never be cleared from the `gauges` mapping. This also prevents the creation of a new gauge for a pool whose gauge was killed totally. To fix this, the report suggests correctly clearing the `gauges` mapping when killing a gauge totally. The bug has been resolved in a recent pull request by completely removing the `killGaugeTotally` function.

## Details
When calling `killGaugeTotally`, the `_gauge` key in the `poolForGauge` mapping is first [deleted](https://github.com/ThenafiBNB/THENA-Contracts/blob/2bb84cf6bb429e9bac8ac8464f4546da855ef4ca/contracts/VoterV3.sol#L322) then later [retrieved](https://github.com/ThenafiBNB/THENA-Contracts/blob/2bb84cf6bb429e9bac8ac8464f4546da855ef4ca/contracts/VoterV3.sol#L327) and used as a key to clear the [`gauges` mapping](https://github.com/ThenafiBNB/THENA-Contracts/blob/2bb84cf6bb429e9bac8ac8464f4546da855ef4ca/contracts/VoterV3.sol#L328). This retrieval will always return the zero address, resulting in the pool address never being cleared from the `gauges` mapping.


Additionally, this results in the inability to [create a new gauge](https://github.com/ThenafiBNB/THENA-Contracts/blob/2bb84cf6bb429e9bac8ac8464f4546da855ef4ca/contracts/VoterV3.sol#L554) for a pool whose gauge was killed totally, as the mapping will already have an [address set for the given pool](https://github.com/ThenafiBNB/THENA-Contracts/blob/2bb84cf6bb429e9bac8ac8464f4546da855ef4ca/contracts/VoterV3.sol#L556).


In order to avoid an invalid gauge state, consider correctly clearing the `gauges` mapping when killing a gauge totally.


***Update:** Resolved in [pull request #3](https://github.com/ThenafiBNB/THENA-Contracts/pull/3) at commit [35c01c8](https://github.com/ThenafiBNB/THENA-Contracts/pull/3/commits/35c01c8709bb61cc893b79f9b425be2a353fcb6e). `killGaugeTotally` has been completely removed.*
