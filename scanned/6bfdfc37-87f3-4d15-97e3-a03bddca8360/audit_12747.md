# External mints affect the number of tokens available for sale in `NFTMintSale`

## Summary
This bug report is about a problem with the `NFTMintSale` contract. When an external minter mints a token, it affects the `nft.totalSupply()` value, reducing the number of available tokens for sale in the contract. To fix this, the report recommends using a local counter of minted tokens within the `NFTMintSale` contract instead of relying on the `nft.totalSupply()` value. This will prevent interference between external mints and the availability of tokens for sale.

## Details
##### Description
If an external minter of `nft` mints a token, it affects the `nft.totalSupply()` value, reducing the number of available tokens for sale in the `NFTMintSale` contract.

##### Recommendation
