# Unchecked Return Value

## Summary
The client has acknowledged a bug that can cause unintended consequences in the protocol. The bug is caused by not checking return values in certain parts of the code, which can indicate the success of a process. The affected files are `BorrowerOperations.sol`, `CollateralManager.sol`, `StETHoracle.sol`, `PriceFeed.sol`, `StabilityPool.sol`, `TroveManager.sol`, `TroveManagerRedemptions.sol`, `EToken.sol`, and `USDEToken.sol`. The recommendation is to check the return values to prevent any potential issues.
