# [H-02] It might not be possible to withdraw tokens from the basket

## Summary
This bug report is about a vulnerability in the Basket Token system. If enough token owners exit, it will be impossible to exit the pool with the last minimum amount of tokens due to a check in the BasketFacet.sol file. This would result in the tokens being locked forever. A recommended mitigation step is to either remove the check or perform it only for the owner balance, who would need to have at least the minimum amount of tokens.

## Details
_Submitted by Czar102, also found by csanuragjain_

#### Impact

When enough basket token owners exit, it will be impossible to exit pool with the last `MIN_AMOUNT` tokens because of [this](https://github.com/code-423n4/2021-12-amun/blob/main/contracts/basket/contracts/facets/Basket/BasketFacet.sol#L228-L231) check. This will result in locking some tokens forever.

#### Recommended Mitigation Steps

Consider resigning from this check or performing it only for the owner balance, who would need to have at least `MIN_AMOUNT` tokens.
