# Anyone can initialize contract at Staking.sol

## Summary
The function initialize() can be accessed by anyone, which can cause unexpected staking start timestamps. To prevent this, it is recommended to use the modifier onlyOwner to restrict the function's use to the owner.

## Details
**Description**

Function initialize() can be called by anyone and if it fit all requires can set unexpected staking
start timestamp.

**Recommendation**:

Use modifier onlyOwner to restrict function use to the owner.
