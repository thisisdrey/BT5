# No method to withdraw stuck ETH

## Summary
This bug report is about a missing method in a contract called `TradableSettingsMessageAdapter`. This contract is used to pay fees for sending messages across different chains. The issue is that there is no way to withdraw any ETH that gets stuck in this contract if it is no longer being used. The recommendation is to add a method for the admin to withdraw this stuck ETH, similar to a method in another contract called `TradableSideVault`. The bug has been resolved and is considered to have a medium severity level.

## Details
**Severity**: Medium

**Status**: Resolved

**Description**

In Contract `TradableSettingsMessageAdapter`, there is no method to withdraw the stuck ETH in case this contract is not used anymore. 

This contract will have ETH as it will be used for paying fees to send messages on across chains using Layer Zero.

Recommendation: Add a method similar to the method `transferGas(...)` in the TradableSideVault contract to transfer stuck ETH only for admin.
