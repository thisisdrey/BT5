# CalculateRequestHash does not include QuorumIDs

## Summary
The bug report is about a medium risk bug in the code located in the file `churner.go` on lines 312 to 318. The function in question hashes everything except for the `QuorumIDs` field. This means that if someone is able to intercept the request, they can change the `QuorumIDs` value. The recommendation is to include the `QuorumIDs` field in the hash to prevent this issue.

## Details
## Severity: Medium Risk

## Context
`operators/churner/churner.go#L312-L318`

## Description
This function hashes everything except `churnRequest.QuorumIDs`. If it's possible to intercept this somehow, an intercepted request could provide a different `QuorumIDs` value.

## Recommendation
Include the `QuorumIDs` field in the hash.
