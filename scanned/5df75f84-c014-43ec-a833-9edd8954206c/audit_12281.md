# Anyone Can Mint an Arbitrary Number of NFTs

## Summary
The client has implemented the recommended actions to fix a bug in the `TGKNFTContract` file. The bug caused the `mintNFT()` function to not restrict which addresses could create NFTs, even though it was supposed to only allow the owner of `TGKNFTContract` to do so. The suggested solution is to add the `onlyOwner` modifier to the `mintNFT()` function.

**Update**
The client implemented the recommended actions.

**Update**
Marked as "Fixed" by the client. Addressed in: `97625a33a9dd4f4cbce659c86c2e59359caf1359`.

**File(s) affected:**`TGKNFTContract.sol`

**Description:** The function `TGKNFTContract.mintNFT()` does not restrict which addresses can mint NFTs since this function is public and has no access control. Per the client's explanation, only the owner of `TGKNFTContract` should be able to call `mintNFT()`.

**Recommendation:** Consider using the `onlyOwner` modifier in `TGKNFTContract` and adding the modifier to the `mintNFT()` function.
