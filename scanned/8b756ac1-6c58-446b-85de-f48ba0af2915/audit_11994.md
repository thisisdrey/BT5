# Rebate Estimator Address Desync

## Summary
The client has marked a bug as "Fixed" by the client. The OasisSwap team has removed the `setRebateEstimator()` function and the `getRebate()` function will now inherit values from the factory contract. The bug affected the `OasisSwapFactory.sol` and `OasisSwapRouter.sol` files. The problem was that both contracts had the `setRebateEstimator()` function, which could lead to different rebate estimator contracts being referred to. This could cause desync issues when calling `getRebate()` from either contract. The recommendation is to remove the `rebateEstimator` variable and its associated functions from the `OasisSwapFactory` contract and switch the `getRebate()` dependency to the router contract within the `swapCalculatingRebate()` function of the `OasisSwapPair` contract.

## Details
**Update**
Marked as "Fixed" by the client. Addressed in: `61d09e20ba5b567e86c0806174237bbbc2c5c498`.

The OasisSwap team removed the `setRebateEstimator()` function. The `getRebate()` function inherit values from the factory contract.

**File(s) affected:**`OasisSwapFactory.sol`, `OasisSwapRouter.sol`

**Description:** The `OasisSwapFactory` and `OasisSwapRouter` contracts both contain the `setRebateEstimator` function, which allows the setting of the rebate estimator address. However, an update in one does not necessitate an update in the other. There is a possible scenario that both `OasisSwapFactory` and `OasisSwapRouter` contracts will refer to different rebate estimator contracts upon reconfiguration. Having ability to configure the rebate estimator address independently at both contract leads to potential desync issues when calling `OasisSwapFactory.getRebate()` versus `OasisSwapRouter.getRebate()`.

**Recommendation:** Remove the `rebateEstimator` variable and its associated getter and setter functions from the `OasisSwapFactory` contract. Switch `getRebate()` dependency from the factory to the router contract within the `swapCalculatingRebate()` function of the `OasisSwapPair` contract.
